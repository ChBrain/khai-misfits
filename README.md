# khai misfits: Misfits

The Misfits production house: a collection of khai plays staged as misfits, each
a production staged with khai, dedicated to Misfits.

This README is the house's **Estate identity**, the production that answers for
the whole run. Every misfit in `misfits/` links this house in its Estate (E); a
misfit with no Estate is not yet a production.

- **`misfits/`** holds the productions, written in khai-playwright mode.
- **how it was built:** raised by khai-stage (the impresario's generator), wired
  to the khai canon and kit, gated, and protected. See `CLAUDE.md`.
