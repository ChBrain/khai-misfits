Add The File Drawer misfit (publication bias / the file-drawer problem). A study is kept for the record not by how well it was done but by whether its result came out significant, so the significant enter the literature and the nulls file into the drawer, and the sample everyone reads and pools is filtered on the very outcome it is read to learn, overstating every effect it holds. No fraud is needed: the author writes up the finding where the citation is, the reviewer prefers the novel, and the editor spends a scarce page where it is read, and the sum of reasonable choices is a record surer than the science. Worse, the replication that is science's own self-correction is a null when it fails, drawered by the same filter, so the false positive stands. Villain-less and remedy-costly: the cure, pre-registration and publishing nulls, pays no author a citation and no editor a reader. Warranted on Sterling (1959), Rosenthal (1979, the file-drawer problem and the fail-safe N), Ioannidis (2005), Dickersin (1990, the medical evidence and the case for registration), and Simmons, Nelson & Simonsohn (2011, the p-hacking cousin, set apart); cut explicitly against The Survivors (an active filter on the result, not passive attrition, the missing evidence drawered rather than destroyed), Metric Fixation (no target set and gamed), and The Streetlight Effect (the bias is in which honest results the record kept, not where anyone searched). Third target of the warrant-mine slate.
