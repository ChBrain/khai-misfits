# Process: Covering

## Taxonomy

[The Paper Trail](play_the_paper_trail.md). Each act done for the record it leaves rather than the outcome it reaches, and the file that grows from it.

## Owner

- Project: khai-misfits

## Initiated by

An incentive already set toward the process, and a hand that reads it right.
The coverer, at [the desk](place_the_desk.md), choosing for each act the version that shields the file, and the auditor certifying it done.

## Direction

Toward a spotless record and away from the aim.
The shielding test is ordered, the covering email sent, the extra sign-off gathered, each act picked for how it reads later, and the auditor stamps every box as ticked, so the file thickens, clean and complete, whatever the outcome.

## Lever

A record that protects the worker whether or not the work succeeded.
Because the documented folly is spared what the undocumented judgement is not, the rational act is the defensible one, so covering pays every time and the paper accretes as fast as the effort it consumes.

## Echo

A file that says everything but whether it worked.
The steps documented, the boxes certified, the record clean and thick, and the effort that built it drawn, act by act, off the outcome and onto the paper.
