# Plot: The Cover

## Taxonomy

[The Paper Trail](play_the_paper_trail.md)

## Owner

- Project: khai-misfits

## Cue

[The desk](place_the_desk.md), each act facing two versions of itself.
One serves the outcome, one shields the file, and the incentive set at the tribunal has already made clear which one protects the hand that does it.

## Action

[Covering](process_covering.md): [The Coverer](persona_the_coverer.md), holding [the coverer's seat](position_coverer.md), orders the shielding test, sends the covering email, gathers the extra sign-off, while [The Auditor](persona_the_auditor.md), holding [the auditor's seat](position_auditor.md), certifies each box as done.

## Stage

[The Coverer](persona_the_coverer.md) tables [Cover Yourself](plan_cover_yourself.md) and makes [the cover](piece_the_cover.md), the act chosen for the file and not the aim.
[The Auditor](persona_the_auditor.md) tables [Tick the Box](plan_tick_the_box.md) and stamps [the record](piece_the_record.md) complete, so the file grows clean and thick.

## Tension

Every act is done right for the one who does it, and the file fills with rational covers.
The coverer is not cutting corners: it is answering the blame exactly as the blame demands, choosing each time the version that leaves it defensible, and the auditor is checking precisely what it was told to check. So the paper accretes, each sheet a correct move, and none of it says whether the work reached the aim, because the record measures the steps and the steps are now the point.
