# Piece: The Someday

## Taxonomy

[The Tyranny of the Urgent](play_the_tyranny_of_the_urgent.md). The important's ever-receding later, the deferability that lets a deadline-less task be postponed each day because one more day never costs much.

## Owner

- Project: khai-misfits

## Place

Recedes through [the backlog](place_the_backlog.md), the later the important is always pushed to.
Would be ended only from [the diary](place_the_diary.md), where a someday is turned into a today by fencing the hour.

## Load Bearing

Without it the important would have to be done or dropped, and there would be no third state to hide in.
With it the deferral has its mechanism: a later that is always available because the important carries no deadline, so it can be postponed today at almost no cost, and again tomorrow, the someday receding exactly as fast as it is approached. The Someday is load bearing because it is the deferability itself (Covey): the perpetual later a deadline-less task can always be moved to, so the piece carries the mechanism by which the worthy work is neither done nor abandoned but endlessly postponed, safe and unreached.

## Apparent

A plan, the reassuring sense that the important is not lost, only scheduled for a better time.
Seen at the surface, the Someday looks like intention kept, the worthy work held for when there is room, and the truth that the room never comes because the urgent fills every day, that the receding later is how a thing is never done, that safe-for-later and never-done are the same place, stays hidden behind how much a someday feels like a promise rather than a loss.

## Yearbook

Kept in [the backlog](place_the_backlog.md) through [deferring](process_deferring.md), the later the important is always pushed to.
It is the mechanism of the trap: the Someday lets [the payoff](piece_the_payoff.md) be postponed without ever being dropped, the receding later that turns each reasonable deferral into [the undone](piece_the_undone.md), the hiding place the [tyranny of the urgent](piece_the_tyranny_of_the_urgent.md) keeps the worthy work in.
