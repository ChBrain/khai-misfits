# Place: The Trend

## Taxonomy

[The New Normal](play_the_new_normal.md). The chart where the anomalies are plotted, and the line is quietly redrawn.

## Owner

- Project: khai-misfits

## Shown

Points over time, each a little further out, and a reference line.
The data that shows the deviation and the level called normal: the signal against the baseline, run after run.

## Holds

The signal, and the drifted baseline it is measured against.
The place the moving reference lives, redrawn to the last accepted point rather than the spec.

## Offers

A picture of the deviation.
Here the anomaly can be seen, tracked, and, if measured against the spec, recognised for what it is.

## Withheld

Which line to read it against.
The trend shows both the points and a reference, and lets the reference be the last success, so the same chart that could reveal the drift is the one that hides it in plain sight.
