# Piece: The Spec

## Taxonomy

[The New Normal](play_the_new_normal.md). The original limit: the boundary the design drew, and the reference the drift stops using.

## Owner

- Project: khai-misfits

## Place

At [the review](place_the_review.md), written into the design and quoted less with every meeting.
The line that says this far and no further: the erosion limit, the tolerance, the margin the system was certified to hold. Fixed, documented, and slowly becoming a number nobody measures against.

## Load Bearing

Without it there is no line for a signal to cross and no fixed thing to drift away from.
With it there is a true reference the whole play is defined against: the Spec is what the first signal violates and what every later one should still be judged by, so the misfit is precisely that judging shifts from the Spec to the last success. It is load bearing because it is the line the drift abandons, and the line the reset restores.

## Apparent

Conservative. Probably over-cautious.
It looks like a limit set with margin to spare, the kind engineers pad for safety, so crossing it a little seems harmless, even sensible, given how much the system keeps surviving past it. That reading is the trap, because the margin the Spec holds is invisible until the day it is the only thing that would have held.

## Yearbook

Set by the design and quoted by [the engineer](persona_the_engineer.md), abandoned as the reference through [normalizing](process_normalizing.md).
It is the line the play is measured against: the Spec is what [the baseline](piece_the_baseline.md) drifts away from, and what [the reset](piece_the_reset.md) puts back at the center of the room.
