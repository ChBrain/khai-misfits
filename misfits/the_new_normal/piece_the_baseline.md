# Piece: The Baseline

## Taxonomy

[The New Normal](play_the_new_normal.md). The drifted boundary: the new normal each later signal is judged against instead of the spec.

## Owner

- Project: khai-misfits

## Place

At [the trend](place_the_trend.md), a line quietly redrawn to wherever the last accepted point sits.
Not the spec and not any single decision, but the moving reference the room now carries in its head: the level of deviation that reads as ordinary, updated each time a larger one survives.

## Load Bearing

Without it each signal would be judged against the fixed spec and the drift could not compound.
With it the reference itself moves, so a signal that would have alarmed anyone against the original line looks routine against the last success, and the system migrates toward the edge of safe performance without any step that looks like a step. The Baseline is load bearing because it is the thing that drifts, the standard that erodes by being redefined rather than lowered.

## Apparent

Experience. Maturity. Knowing what is really fine.
It looks like hard-won judgment: the team has seen this before, knows what actually matters, has calibrated its concern to reality. That reading is the disguise, because the calibration is to survivals only, and a baseline set by what has not broken yet is a baseline that can only move one way.

## Yearbook

Shifted by every [reclassification](piece_the_reclassification.md) through [normalizing](process_normalizing.md), away from [the spec](piece_the_spec.md).
It is the moving line the play tracks: the Baseline is what [the veteran](persona_the_veteran.md) defends as wisdom, and what [the reset](piece_the_reset.md) must drag back to the design limit.
