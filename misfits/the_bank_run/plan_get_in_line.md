# Plan: Get in Line

## Taxonomy

A **standing plan**: the intent of [the depositor's seat](position_depositor.md), inside [The Bank Run](play_the_bank_run.md).

## Owner

- Owner: [Depositor](position_depositor.md)

## Direction

Watch the crowd, not the books. If a line may form, be near its front, because the till is finite and the last in it gets nothing. Withdraw early and in full, and do not be the one left holding a claim on an empty window.

## Orders

([The Depositor](persona_the_depositor.md) reckons that if the others run he must run first, and joins the line.)

## Implementation

([The Window](place_the_window.md); the [running](process_running.md) process; [the queue](piece_the_queue.md) makes running first the right reply, and each right reply feeds [the self-fulfilling prophecy](piece_the_self_fulfilling_prophecy.md).)

## Targets

- [ ] the chance of a run is weighed against the cost of being last
- [ ] the front of the line is reached early and the claim withdrawn in full
- [ ] the withdrawal helps make real the shortfall it withdrew to escape
- [ ] every saver's sound reply to an expected run is one more runner in it
