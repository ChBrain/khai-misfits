# Position: Doubter

## Taxonomy

[The Bank Run](play_the_bank_run.md). The seat that carries the belief the bank may not be able to pay.

## Owner

- Project: khai-misfits

## Has

A doubt, well founded or not, about whether the bank can pay.
An audience of savers who might act on it.
No need to be correct, only to be believed by enough.

## Orders

Pass the doubt along.
Turn a private worry into a shared expectation.
Set the others reckoning with the crowd instead of the loans.

## Loses

Nothing, and is confirmed either way.
If the doubt spreads it comes true, and the seat is proved right by the run it started; if it does not, it costs the seat nothing to have wondered aloud.
The one seat that wins whether or not it was ever correct.

## Drives

Toward the shared belief in a failure,
away from any test of whether the failure is real,
so a prophecy is set loose that needs only enough listeners to make itself true.
