# Plot: Fast Path

## Taxonomy

[The Bus Factor](play_the_bus_factor.md)

## Owner

- Project: khai-misfits

## Cue

[The keyboard](place_the_keyboard.md), where the system was built and is best understood.
A critical thing rests on one person's unwritten know-how. Through [the queue](place_the_queue.md) the questions are already starting to arrive, because everyone knows exactly who to ask.

## Action

[Routing](process_routing.md): [The Keystone](persona_the_keystone.md), holding [the keeper's seat](position_keeper.md), answers each question directly, because it is faster than teaching and gladly given.
[The Asker](persona_the_asker.md) at [the team member's seat](position_team_member.md) sends the next one straight to the same head, for the same reason.

## Stage

[The Keystone](persona_the_keystone.md) tables [Just Ask Me](plan_just_ask_me.md): resolve it now, in person, and [the tacit knowledge](piece_the_tacit_knowledge.md) stays exactly where it was learned.
[The Asker](persona_the_asker.md) tables [Ask the Expert](plan_ask_the_expert.md): take the quick route, and [the fast path](piece_the_fast_path.md) forms, a lesson given and never recorded.

## Tension

Nobody is hoarding.
Answering directly is generous and quick, asking the person who knows is sensible, and writing it all down first would be absurdly slow. But the quick way is the way that leaves no copy, so the very helpfulness of the exchange is what keeps the knowledge in one head, and the fast path is set before anyone notices it is also the path that does not share.
