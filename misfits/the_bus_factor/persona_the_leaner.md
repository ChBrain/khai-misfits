# Persona: The Leaner

## Taxonomy

[Delivery Manager](position_delivery_manager.md). Focused on the date, and the date always lands.

## Owner

- Project: khai-misfits

## Projection

Focused. Delivery-driven. Hits the dates.
The board moves, sprint over sprint, clean.
Whatever ships this week, ships this week.

## Action

Pulls delivery every sprint, and lets the documentation wait.
There is work that ships and work that does not, and shipping wins.
The docs card slides to the next column, always.

## Shadow

Shipping beats documenting when there is a deadline, and there is always a deadline.
So the reasonable this-sprint choice gets made forever... the one task that would break the trap deferred, sprint after sprint, by a manager doing exactly the job.
No villain here, only a backlog that never surfaces the thing that matters most.

## Tell

Says "docs next sprint, we need to ship," and ships.
Next sprint, says it again.
