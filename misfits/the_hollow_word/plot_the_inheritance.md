# Plot: The Inheritance

## Taxonomy

[The Hollow Word](play_the_hollow_word.md)

## Owner

- Project: khai-misfits

## Cue

[The Currency](place_the_currency.md), a newcomer arriving after the practice has gone.
The Newcomer learning the word as coin, [the ground](place_the_ground.md) standing empty where the referent was.

## Action

[Inheriting](process_inheriting.md): [The Newcomer](persona_the_newcomer.md), holding [the newcomer's seat](position_newcomer.md), learns the word already hollow, takes the current use for the whole content, and passes the token on.

## Stage

The [newcomer](persona_the_newcomer.md), on [Learn the Word](plan_learn_the_word.md), receives [the word](piece_the_word.md) as [the ghost](piece_the_ghost.md), the residual sense the only content on offer, and hands it to the next arrival.
Never having seen [the referent](piece_the_referent.md), the newcomer cannot miss it, so the hollow is learned as coin and carried forward.

## Tension

Learning a living word from its current use is the only way there is to learn it, and the honest learning is what seals the drift.
There is nothing to fault: the practice is gone before the newcomer arrives and there is no fuller sense to learn. But a token is all there is to inherit, so the plot closes with the emptied word passed to the next speaker, the currency kept alive over an absent referent and the drift sealed by good faith.
