# Position: The Vacancy

## Taxonomy

[The Sort](play_the_sort.md). The open seat, a spot left empty by a departure and waiting to be filled, which cannot be occupied without shifting the block it sits in.

## Owner

- Project: khai-misfits

## Has

An opening, and a pull on whoever is looking.
An empty spot the market will fill: the seat cannot stay vacant indefinitely, so it draws an occupant, and the occupant it draws is whoever finds that block a fit.
No neutral filling available, since every candidate occupant is of one kind or the other and each arrival moves the local mix, so the seat cannot be filled without tilting the block one way.

## Orders

Settle where I fit.
Draw the occupant for whom this block suits, since a vacancy in a block already tilting toward one kind is a fit for more of that kind, and offer them the opening.
Fill, because an empty spot is unstable and the neighbourhood resolves it by occupancy, so the seat's function is to be taken, not to be held open.

## Loses

Neutrality, which no filled seat can keep.
A vacancy opened by a departure sits in a block already a little more one-sided, so the occupant it fits is of the kind that departure left behind, and filling the seat with them tilts the block further, drawing the next vacancy toward the same kind.
What the seat forfeits is any settling that does not propagate: because it can be filled only by someone, and someone is always of one kind, the Vacancy converts a departure into a further tilt no matter who takes it, so the opening that looks like a free choice for whoever arrives is a step in the sort whichever way it is filled.

## Drives

Toward a filling that fits the tilt,
and a tilt the filling deepens,
so the seat that must be occupied by someone is the one that turns each departure's opening into the next kind's arrival.
