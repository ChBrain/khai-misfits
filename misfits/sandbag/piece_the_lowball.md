# Piece: The Lowball

## Taxonomy

[Sandbag](play_sandbag.md). The modest commit talked down before the target is set.

## Owner

- Project: khai-misfits

## Place

At [the forecast](place_the_forecast.md), in the careful, gloomy case made when next year's number is up for negotiation.
The commit argued down at the outset: the headwinds stressed, the pipeline played modest, the ask lowered before it hardens into a target that will bind for a year.

## Load Bearing

Without it the target is set from an honest estimate and the plan reflects the world.
With it the target is set from a negotiation, because the Lowball is sandbagging moved to planning time: rather than beat a number and pay for it later, the forecaster shapes the number downward before it is fixed, hoarding slack inside the commit so the year starts with room to spare. It is the same instinct as holding output back, applied one step earlier, and it corrupts the plan at its source, since the figure the whole outfit steers by is chosen to be beatable, not to be true.

## Apparent

Prudent, conservative forecasting.
It looks like good sense and healthy caution, a manager declining to over-promise, under-promising so as to over-deliver. The framing is respectable and half-sincere, which is what makes it work: no one can prove a modest forecast was a lowball rather than an honest read, so the slack goes in unchallenged and the target lands soft.

## Yearbook

Tabled by [the lowballer](persona_the_lowballer.md) through [negotiation](process_negotiation.md), and undone only when the target stops being set from the seat's own promise.
It is the play's opening gambit: the Lowball builds the slack in before the year begins, so [the hidden capacity](piece_the_hidden_capacity.md) starts the race already parked and the number is soft from the first day.
