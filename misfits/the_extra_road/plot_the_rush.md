# Plot: The Rush

## Taxonomy

[The Extra Road](play_the_extra_road.md)

## Owner

- Project: khai-misfits

## Cue

[The map](place_the_map.md), and a routing service that sends everyone the same way at once.
The shortcut wins for each, so the model names it quickest for all, and the trickle onto the link becomes a flood.

## Action

[Equilibration](process_equilibration.md): [The Dispatcher](persona_the_dispatcher.md), holding [the dispatcher's seat](position_dispatcher.md), routes the whole fleet by shortest path, and the last drivers on [the old route](place_the_old_route.md) drop onto the link to keep up.

## Stage

[The Dispatcher](persona_the_dispatcher.md) tables [Route the Fleet](plan_route_the_fleet.md): optimise each trip in isolation rather than the whole, so [the selfish route](piece_the_selfish_route.md) becomes an instruction at scale and [the equilibrium](piece_the_equilibrium.md) hardens into place.
The last drivers give up [the old route](place_the_old_route.md), and the bad balance settles fully in.

## Tension

The network reaches a state no one can beat by switching, and no one meant to build.
When one service routes every driver the same way, the selfish route stops being an accident and becomes coordinated, and the last drivers leaving the old route remove the final drag on the slow equilibrium. Everyone would gain if many moved back together, and no one gains by moving alone, so the flows lock ... optimal for each trip, and settled exactly at the slow point.
