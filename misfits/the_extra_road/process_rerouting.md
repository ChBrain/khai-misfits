# Process: Rerouting

## Taxonomy

[The Extra Road](play_the_extra_road.md). The shift onto the new link once it opens.

## Owner

- Project: khai-misfits

## Initiated by

A faster road appearing where there was none.
The extra link opening, and the first drivers turning onto it.

## Direction

The traffic moving off the old routes and onto the shortcut.
Each reroute rational, and each adding to the load the next one meets.

## Lever

The dominance of the new link for any single driver.
A route better for each, so taken by all.

## Echo

The shared stretch loaded, the old balance abandoned.
The network re-poured onto the very link that will slow it.
