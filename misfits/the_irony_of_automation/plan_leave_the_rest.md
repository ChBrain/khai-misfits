# Plan: Leave the Rest

## Taxonomy

A **standing plan**: the intent of [the designer's seat](position_designer.md), inside [The Irony of Automation](play_the_irony_of_automation.md).

## Owner

- Owner: [Designer](position_designer.md)

## Direction

Automate every task that can be automated, and assign the human whatever is left. Build the machine to handle all the tractable, routine cases, trust the operator with the residue, and read the automated system as the job done well.

## Orders

([The Designer](persona_the_designer.md) mechanises the tractable and leaves the intractable to the operator, expecting the human to handle in the moment what the design could not.)

## Implementation

([The Floor](place_the_floor.md); the [automating](process_automating.md) process; automating [the routine](piece_the_routine.md) leaves the human the hardest residue, delivered later as [the cold seat](piece_the_cold_seat.md).)

## Targets

- [ ] every task that can be automated is automated
- [ ] the residue too hard or rare to mechanise is left to the operator
- [ ] the human is trusted with exactly what the design could not do
- [ ] the automated system is read as the job done well
