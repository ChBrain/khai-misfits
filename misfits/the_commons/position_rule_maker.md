# Position: Rule-Maker

## Taxonomy

[The Commons](play_the_commons.md). The seat that writes the cap.

## Owner

- Project: khai-misfits

## Has

The pen, and the paper the limit is written on.
The wise limit itself, the number all would keep if all kept it.

## Orders

Draft the number the green can bear.
Write the rule each would follow, if only the others followed too.
Publish it, fair and plain.

## Loses

The rule to non-compliance: a cap is only words until it is kept.
Agreement without the teeth to hold it, a promise everyone breaks alone.

## Drives

Toward a cap, sensible and shared.
Away from a cap that binds, for the pen writes no penalty.
The wise suggestion overgrazed, ignored by the very herds it would save.
