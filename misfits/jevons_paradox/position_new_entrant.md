# Position: New Entrant

## Taxonomy

[Jevons Paradox](play_jevons_paradox.md). The seat that finds uses the old price never allowed.

## Owner

- Project: khai-misfits

## Has

An eye for a service suddenly cheap enough to build on.
The freedom to put the resource where it could never afford to go.
Real value to create at the new price.

## Orders

Open the uses the old price ruled out.
Build on the cheaper service and grow the market for it.
Turn a saving into a whole new demand, legitimately and well.

## Loses

Nothing it can see, because its gains are real.
The new uses are genuine value, and they are also consumption the projection never counted, added to a set of uses it assumed was fixed.
Grows the whole precisely by inventing more to spend it on.

## Drives

Toward new value the old price forbade,
which is real and good and also more resource used,
so the extensive margin opens exactly as the efficiency was meant to close it.
