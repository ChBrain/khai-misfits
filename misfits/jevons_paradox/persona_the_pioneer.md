# Persona: The Pioneer

## Taxonomy

[New Entrant](position_new_entrant.md). The one who finds uses the old price never allowed.

## Owner

- Project: khai-misfits

## Projection

Enterprising, inventive, alert to opportunity.
Sees a service suddenly cheap enough to build on, and builds.
Creates real value nobody could afford before.

## Action

Opens new uses that were never viable at the old price.
Puts the cheaper resource to work in places it could never have gone.
Grows the demand not by using more of the same but by inventing more to use.

## Shadow

Its new value is new consumption the saving never counted.
The projection subtracted from a fixed set of uses, and the Pioneer adds uses to the set, so the resource is spent on things that did not exist to be spent on before.
Genuinely creative, and genuinely why the meter climbs past every forecast.

## Tell

"At this price, you could even use it for that."
And now that is another draw on the resource that was meant to shrink.
