# Piece: The New Use

## Taxonomy

[Jevons Paradox](play_jevons_paradox.md). The extensive margin: uses that were never viable at the old price and become so at the new.

## Owner

- Project: khai-misfits

## Place

At [the market](place_the_market.md), where a service cheap enough to build on gets built on.
In the applications that could not have paid before, and now pay.

## Load Bearing

Without it the rebound would be bounded by the uses that already existed, and might stay under the saving.
With it the set of uses itself grows: a resource cheap enough opens doors that were shut, so demand rises not only in the old uses but in wholly new ones the projection never counted. The New Use is load bearing because it is the extensive margin (Sorrell), the reason rebound can exceed the gain and tip into backfire, since you cannot bound the new consumption by the old when the cheapness invents demand that did not exist.

## Apparent

Innovation, and genuine value.
It looks like progress of the best kind: a cheaper resource enabling things nobody could afford before, real goods, real invention. And it is, which is why the new consumption it brings is welcomed rather than counted against the saving.

## Yearbook

Opened by [the pioneer](persona_the_pioneer.md) through [rebounding](process_rebounding.md), unpriced by [the expected saving](piece_the_expected_saving.md).
It is the rebound's second engine: the New Use adds to the very set of uses the forecast held fixed, so with [the rebound](piece_the_rebound.md) it pushes the total past the saving and into [the backfire](piece_the_backfire.md).
