# Plot: The Return Path

## Taxonomy

[The Decision Travels](play_the_decision_travels.md)

## Owner

- Project: khai-misfits

## Cue

[The room where it lives](place_the_room_where_it_lives.md), and someone who has stopped filing symptoms and started reading them together.
The question has changed: not what is the one thing we can log today, but what does this room actually hold, and how does it reach the desk that could still change the drawing.

## Action

[Integration](process_integration.md): [The Witness](persona_the_witness.md), given a real channel and holding [the integrator's seat](position_integrator.md), reads the fragments as one whole and carries the room's reality back up the chain.

## Stage

[The Witness](persona_the_witness.md) tables [Carry It Back](plan_carry_it_back.md): run the chain in reverse rather than close another ticket, so [the traverse](piece_the_traverse.md) is named whole and sent to a station that can still act, and the site is measured before the next tolerance locks.
With the room's reality upstream, the governing variable is touched at last, and the accumulation stops accumulating: the order is the story.

## Tension

The loop closes, and the clean forward-only chain that ran so smoothly is gone.
Turning the chain around works, and it costs: it gives up the settled basis every downstream station inherited, asks the first desk to reopen a decision it had signed, and trades a fast, defensible order for a slow, contested one that keeps having to go back. The room gets a desk ... or the return path is judged too slow and too dear, the chain resumes filing one symptom at a time, and the decision travels on as it always has.
