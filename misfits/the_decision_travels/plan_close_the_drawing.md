# Plan: Close the Drawing

## Taxonomy

A **standing plan**: the intent of [the design engineer's seat](position_design_engineer.md), inside [The Decision Travels](play_the_decision_travels.md).

## Owner

- Owner: [Design Engineer](position_design_engineer.md)

## Direction

Turn the named category into a buildable drawing that answers every requirement.
Completeness is the whole intent: a sheet with no open question, correct by every measure the office can apply.

## Orders

[The Architect](persona_the_architect.md) traces each requirement to a feature, justifies each tolerance, and signs the drawing only when nothing is left undefined.

## Implementation

[The design office](place_the_design_office.md); the [R&D](process_rd.md) process; the category as the only description of the user the seat is ever given.

## Targets

- [ ] the category is rendered into geometry
- [ ] every requirement is traced to a feature
- [ ] every tolerance is justified
- [ ] the drawing is closed and handed down
