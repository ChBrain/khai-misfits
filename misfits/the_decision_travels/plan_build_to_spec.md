# Plan: Build to Spec

## Taxonomy

A **standing plan**: the intent of [the production manager's seat](position_production_manager.md), inside [The Decision Travels](play_the_decision_travels.md).

## Owner

- Owner: [Production Manager](position_production_manager.md)

## Direction

Produce a part that matches the drawing inside tolerance, on schedule, and passes every check the drawing defined.
Conformance is the whole intent: build exactly what was specified, and build it well.

## Orders

[The Builder](persona_the_builder.md) sets the line to the drawing, holds the tolerance band, passes conformance, and ships only parts that match.

## Implementation

[The factory floor](place_the_factory_floor.md); the [manufacturing](process_manufacturing.md) process; the tolerance band as it was set upstream, not as the room would have set it.

## Targets

- [ ] the line is set to the drawing
- [ ] parts are produced inside the tolerance band
- [ ] conformance is passed
- [ ] the product ships on schedule
