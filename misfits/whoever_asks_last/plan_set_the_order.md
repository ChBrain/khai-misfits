# Plan: Set the Order

## Taxonomy

A **standing plan**: the intent of [the Rostrum](position_the_rostrum.md), inside [Whoever Asks Last](play_whoever_asks_last.md).

## Owner

- Owner: [The Rostrum](position_the_rostrum.md)

## Direction

Choose a sequence for the contests and run the options a pair at a time, deciding which two meet first and which the survivor faces next, because the choice cannot be taken all at once and some order must be set for the vote to conclude. Keep every step fair, counting each pair honestly, and read the procedure as neutral because no vote in it is miscast, holding that a fairly run sequence yields a fair result. Declare the option left standing the winner and hand it to the room as the room's own decision, since the plan's business is to reach a conclusion and not to prefer one, and it means only to run the meeting to an honest end, blind to the fact that the order it set is the thing that chose.

## Orders

([The Presider](persona_the_presider.md) sets the sequence of pairings, runs them fairly to a single survivor, and declares it the room's choice, holding no preference and meaning only to conclude the vote.)
