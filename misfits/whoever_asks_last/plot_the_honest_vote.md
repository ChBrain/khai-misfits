# Plot: The Honest Vote

## Taxonomy

[Whoever Asks Last](play_whoever_asks_last.md)

## Owner

- Project: khai-misfits

## Cue

[The Division](place_the_division.md), the room taking up its choice.
Options to be decided among, and members each with a settled, honest sense of what they prefer to what.

## Action

[Ranking](process_ranking.md): each member forms a complete and consistent ordering of the options and reports it straight, casting a sincere ballot with no strategy in it.

## Stage

[The Voter](persona_the_voter.md) acts from [the Say](position_the_say.md) on [Rank Them True](plan_rank_them_true.md), supplying [the cycle](piece_the_cycle.md) the honest inputs it will be built from, every ballot sound and fairly weighed.
The vote is as clean as a vote can be, and nothing has gone wrong in any part of it.

## Tension

The vote is faultless, and the faultlessness is where the trouble will hide.
There is no error here: each ranking is consistent, each report is honest, each ballot is counted alike, so the room does exactly what a fair decision asks, and a lesser vote would be the one to distrust. But the soundness is all in the parts, and the parts are about to be summed, so the plot closes with a full set of sincere ballots and a room sure it has done the democratic thing, the inputs beyond reproach and the incoherence they will compose already present in them, unseen because nothing in an honest ranking shows how it will add.
