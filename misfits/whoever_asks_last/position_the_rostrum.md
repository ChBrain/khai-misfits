# Position: The Rostrum

## Taxonomy

[Whoever Asks Last](play_whoever_asks_last.md). The presiding seat, which sets the order in which the options are put to the vote and must return a winner, and finds every order it can set equally fair and each returning a different one, so its neutral duty is a decisive act.

## Owner

- Project: khai-misfits

## Has

The order of the business, and the duty to find a winner.
The seat holds the power to decide which options are put against which and in what sequence, and the charge to run the pairings through to a single survivor, so what it has is the agenda and the obligation to conclude.
It holds the room's trust that the procedure is fair, since every vote it calls is counted honestly and every step is defensible on its own.

## Orders

Set the order, run the pairings, declare the survivor.
Choose a sequence for the contests, take the options a pair at a time because they cannot be taken all at once, and name the one left standing, since a decision must be reached and the seat is what reaches it.
Keep every step fair, because the seat's legitimacy is that no vote in the sequence is miscounted, whatever the sequence is.

## Loses

The neutral order, which does not exist.
The Rostrum can set a fair order and cannot set a neutral one, because when the majority cycles every ordering is equally fair and each selects a different winner, so there is no sequence available that leaves the outcome to the room rather than to the seat. What the position forfeits is a choice that is not its own doing: it must pick an order, the order picks the winner, and no impartiality removes its hand from the result, since impartiality does not single out one sequence. It loses the option of declining, too, because a group that sets no order decides nothing, so the seat cannot put the decisive power down, only exercise it, holding a lever it cannot make idle and cannot pull without authoring the outcome the room could not author itself.

## Drives

Toward a hand that must choose the order and cannot choose it neutrally,
and a decisive power that answers to no preference of the seat and cannot be declined,
so the position asked only to run the meeting fairly is the position that settles what the meeting could not, its every fair order a different verdict and none of them the room's.
