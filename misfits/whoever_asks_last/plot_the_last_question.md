# Plot: The Last Question

## Taxonomy

[Whoever Asks Last](play_whoever_asks_last.md)

## Owner

- Project: khai-misfits

## Cue

[The Order Paper](place_the_order_paper.md), a winner that must still be found.
A cycle with no head, and a decision that has to be reached by taking the options a pair at a time.

## Action

[Sequencing](process_sequencing.md): the Presider sets the order of the pairings, which pair meets first and which the survivor faces last, and the sequence selects the winner, the same ballots returning a different one for every running order.

## Stage

[The Presider](persona_the_presider.md) acts from [the Rostrum](position_the_rostrum.md) on [Set the Order](plan_set_the_order.md), turning [the cycle](piece_the_cycle.md) into a winner by [the sequence](piece_the_sequence.md) they set, authoring [the made majority](piece_the_made_majority.md) without choosing it.
The order reaches a result, and the result is the order's and not the room's.

## Tension

The one move that finds a winner is the move that chooses one, and there is no order that finds without choosing.
There is no rigging: a decision cannot be taken all at once, so some sequence must be set, and every sequence is as fair as every other, counting each pair honestly. But in a ring the survivor is whatever the order shields until last, so the plot closes with a winner fixed by the running order and handed to the room as its own, the Presider holding a decisive power they cannot exercise neutrally and cannot decline, the outcome legitimate in every step and authored by the schedule, whoever asked last having chosen for them all.
