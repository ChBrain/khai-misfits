# Plot: The Adoption

## Taxonomy

[The Done Thing](play_the_done_thing.md)

## Owner

- Project: khai-misfits

## Cue

[The Field](place_the_field.md), an organisation deciding what to become and the recognised form on display.
The Adopter reading the shape the credible peers have and the funders expect, and moving to install it.

## Action

[Adopting](process_adopting.md): [The Adopter](persona_the_adopter.md), holding [the adopter's seat](position_adopter.md), installs the recognised form to be legible to the field it depends on.

## Stage

[The Adopter](persona_the_adopter.md), on [Take the Form](plan_take_the_form.md), stands up [the form](piece_the_form.md), the office and the function and the framework every credible peer has, because the field reads that shape as proper.
The form is taken for the standing it confers, whether or not it fits the work, and the Adopter becomes one more organisation that has it.

## Tension

Adopting the recognised form is prudent, and adopting it is what takes the shape built for standing over the shape that would fit the work.
There is nothing foolish: the field's recognition is real funding and survival. But the form is chosen for the audience that grants legitimacy and not the labour it must organise, so the plot closes with the recognised shape installed, the fit set aside, and one more body added to a convergence the Adopter did not choose and cannot see from inside.
