# Plot: The Slip

## Taxonomy

[The Simpler Lever](play_the_simpler_lever.md)

## Owner

- Project: khai-misfits

## Cue

[The Console](place_the_console.md), a disturbance with no matching response reaching the outcome.
The Regulator running exactly as designed, reaching into its repertoire for a state that is no longer there.

## Action

[Regulating](process_regulating.md): [The Regulator](persona_the_regulator.md), holding [the regulator's seat](position_regulator.md), meets the disturbance with the only thing it can, and finds no response in the trimmed menu to cancel it.

## Stage

[The Regulator](persona_the_regulator.md), on [Hold Steady](plan_hold_steady.md), has no state for this [disturbance](piece_the_disturbance.md), so it passes through uncancelled and the outcome moves, [the shortfall](piece_the_shortfall.md) expressed as [the loss of control](piece_the_loss_of_control.md).
No component has failed: every response still kept works, the regulator runs as designed, and none of the states remaining is the one now needed.

## Tension

The outcome slips, and nothing broke to make it slip.
The control was given away at the cut, not here: the regulator is faithful and the field is ordinary, and the one missing response is the one the trim removed as clutter. So the plot closes with the outcome no longer held and no fault to point to, the loss that was silent at the cut made plain at last, the simpler lever exposed as too small for its world exactly when the world does the thing it no longer answers.
