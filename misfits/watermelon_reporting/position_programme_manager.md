# Position: Programme Manager

## Taxonomy

[Watermelon Reporting](play_watermelon_reporting.md). The rung that aggregates projects into a programme.

## Owner

- Project: khai-misfits

## Has

A portfolio of projects and the authority to report the programme as a whole.
Whoever holds this desk sets the level at which the red is counted, and counts it in the aggregate.

## Orders

Deliver the programme. Balance the portfolio. Report the whole.
The orders are about the aggregate, and in the aggregate one red project is a rounding error.

## Loses

Sight of the task, by design.
The seat is accountable for a level at which a single red has already been averaged away, so it does not lose the red dishonestly: it never held it.

## Drives

Toward a healthy aggregate.
One red netted against many greens keeps the programme number where it needs to be, so the seat is driven to net rather than to surface.
