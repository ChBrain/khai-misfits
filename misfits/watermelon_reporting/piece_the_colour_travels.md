# Piece: The Colour Travels

## Taxonomy

[Watermelon Reporting](play_watermelon_reporting.md). The up axis: the status that greens at every rung.

## Owner

## Place

Up the ladder, rung to rung.
Task to phase to project to programme to portfolio, greening one notch at each hand.

## Load Bearing

Without it the red would arrive at the top as red.
With it the same status climbs and lightens: [the mitigated amber](piece_the_mitigated_amber.md), then [the rounded green](piece_the_rounded_green.md), then [the net green](piece_the_net_green.md), one colour rounded four times.

## Apparent

A clean chain of reports.
Each line reads true and each hand looks like ordinary rolling-up, so the greening is invisible unless you read the rungs together, which nobody upward is paid to do.
