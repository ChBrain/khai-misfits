# Persona: The Mitigator

## Taxonomy

[Phase Lead](position_phase_lead.md). The first rung the colour climbs.

## Owner

- Project: khai-misfits

## Projection

Calm and constructive.
Never denies the red: reframes it. There is always a plan, and a plan turns a colour.
The most reasonable voice in the room, which is what makes the rounding stick.

## Action

Takes the red task and reports the phase amber, because amber says under control.
Attaches a mitigation and an owner and a date, and the attachment is what lets a red task sit inside an amber phase without anyone feeling they rounded.

## Shadow

Redo blows the phase; mitigation saves it.
So the phase is never stopped, only managed, and the difference between managed and fixed is exactly the red the Doer is still carrying.

## Tell

Says "it is amber, we have a plan" and believes it.
Names the external problem out loud: the phase would be green if not for the dependency, so the dependency, not the phase, is red.
