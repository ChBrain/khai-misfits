# Plot: Red Task

## Taxonomy

[Watermelon Reporting](play_watermelon_reporting.md)

## Owner

- Project: khai-misfits

## Cue

[The workface](place_the_workface.md), on the critical path.
A task is red and the float is already spent. The next task is waiting for the hand-off. Nobody above the desk has been asked to stop yet.

## Action

[Execution](process_execution.md): [The Doer](persona_the_doer.md), holding [the task owner's seat](position_task_owner.md), reports the true colour and asks the chain to stop and resource it.
Alongside, [The Neighbour](persona_the_neighbour.md) at [the second doer's seat](position_second_doer.md) waits on the input and would rather take it red than wait.

## Stage

[The Doer](persona_the_doer.md) tables [Get Real Help](plan_get_real_help.md): the one plan that wants the line held, and the one with no leverage to hold it.
[The Neighbour](persona_the_neighbour.md) tables [Take the Hand-off](plan_take_the_handoff.md): keep my own dates, take the work in whatever state it comes.
Between them sits [the red task](piece_the_red_task.md): the honest colour, before any rounding.

## Tension

Nothing here is a lie.
The red is real and the request for help is real, and there is no seat above that is paid to grant it. The colour is already set, and the only seat that wants to stop is the only seat that cannot.
