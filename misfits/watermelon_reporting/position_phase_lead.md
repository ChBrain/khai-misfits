# Position: Phase Lead

## Taxonomy

[Watermelon Reporting](play_watermelon_reporting.md). The first rung of the ladder, where tasks roll up into a phase.

## Owner

- Project: khai-misfits

## Has

A phase, a schedule, and the authority to set the phase colour.
Whoever holds this desk decides how the red tasks beneath it are represented as one status.

## Orders

Keep the phase on schedule. Manage the risks. Report the colour.
The orders reward a phase that stays under control, and a mitigation is how control is shown.

## Loses

The distance between managed and fixed.
The seat can hold a red task inside an amber phase indefinitely, and never has to close that distance, because the phase closing on time is what it is measured on.

## Drives

Toward closing the phase on plan.
Redo blows the plan and mitigation preserves it, so the seat is driven to reframe the red, not resolve it.
