# Plot: Colour Travels

## Taxonomy

[Watermelon Reporting](play_watermelon_reporting.md)

## Owner

- Project: khai-misfits

## Cue

[The stand-up](place_the_stand_up.md), then [the status meeting](place_the_status_meeting.md).
The status leaves the workface and begins to climb. The work, meanwhile, continues sideways, task to task, on a path that was never stopped.

## Action

[Roll-up](process_roll_up.md): [The Mitigator](persona_the_mitigator.md), holding [the phase lead's seat](position_phase_lead.md), reports the phase amber with a plan attached.
Then [consolidation](process_consolidation.md): [The Rounder](persona_the_rounder.md) at [the project manager's seat](position_project_manager.md) rounds amber to green and moves the red into the register.

## Stage

[The Mitigator](persona_the_mitigator.md) runs [Mitigate, Don't Stop](plan_mitigate_dont_stop.md) and hands up [the mitigated amber](piece_the_mitigated_amber.md).
[The Rounder](persona_the_rounder.md) runs [Hold the RAG](plan_hold_the_rag.md) and hands up [the rounded green](piece_the_rounded_green.md).
Both lean on the same device, [the external problem](piece_the_external_problem.md): it is red because of a dependency, so the phase, the project, are not the red thing.

## Tension

Each rounding is fair and each is defensible, because there really is an X.
[The colour travels](piece_the_colour_travels.md) upward one notch at a time while the work travels sideways unchanged, and the two axes never meet, so the green climbs on a task that is still red.
