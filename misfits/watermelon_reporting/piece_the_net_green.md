# Piece: The Net Green

## Taxonomy

[Watermelon Reporting](play_watermelon_reporting.md). The third rounding: one red averaged into a healthy aggregate.

## Owner

- Project: khai-misfits

## Place

[The programme board](place_the_programme_board.md), and up to [the portfolio review](place_the_portfolio_review.md).
The colour of the whole, in which a single project can no longer be seen.

## Load Bearing

Without it a red project would show at the top as red.
With it the red is diluted by scale: counted honestly at a level where one project does not move the number, and so counted into invisibility.

## Apparent

Strategic.
It looks like perspective, the true observation that the programme is bigger than any project, which is exactly what makes the red stop mattering.
