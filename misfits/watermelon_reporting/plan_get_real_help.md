# Plan: Get Real Help

## Taxonomy

A **standing plan**: the intent of [the task owner's seat](position_task_owner.md), inside [Watermelon Reporting](play_watermelon_reporting.md).

## Owner

## Direction

Stop the red where it is, and get the task actually fixed.
This is the only standing plan whose object is repair. It wants the line held, not moved, until the work is right.

## Orders

[The Doer](persona_the_doer.md) reports the true colour, names what the task needs, and asks the chain above to stop and resource it.

## Implementation

[The workface](place_the_workface.md); the [execution](process_execution.md) process; a stand-up whose colour is decided before the Doer speaks.

## Targets

- [ ] the true colour is reported
- [ ] the specific help is named

- [ ] the request is starved, because no seat above is paid to grant it
