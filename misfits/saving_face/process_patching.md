# Process: Patching

## Taxonomy

[Saving Face](play_saving_face.md). Correcting the symptom inside a frame that is never opened.

## Owner

- Project: khai-misfits

## Initiated by

The fixer handed a failure to close.
A queue of faults, each solvable as given.
The reasonable urge to restore the thing and move on.

## Direction

The symptom cleared, fast and real, and the frame left standing.
The recurrence met as new each time, its pattern unnamed.
A closed ticket where a questioned design was due.

## Lever

Craft aimed one loop too shallow: the fix that works, on the wrong layer.
A frame treated as settled because opening it would embarrass its makers.
The thanks each clean fix earns, and the cause each clean fix hides.

## Echo

A system that works again, and fails again, the same way.
Single-loop where double was needed: the symptom served, the cause spared.
The better the patch, the safer the fault beneath it.
