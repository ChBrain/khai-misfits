# Plan: Fall Away

## Taxonomy

A **standing plan**: the intent of [the Margin](position_the_margin.md), inside [The Last Layer](play_the_last_layer.md).

## Owner

- Owner: [The Margin](position_the_margin.md)

## Direction

Weigh the levy against what the levy buys, and support the apparatus only while it returns order and defence worth the surplus given up. When the levy climbs and the return thins to nothing, withdraw: take the surplus to a nearer power, a smaller order, or nothing beyond the base itself, because nothing is owed to a structure that takes more than it gives. The leaving is not rebellion but arithmetic, the same weighing anyone does between a cost and a return.

## Orders

([The Provincial](persona_the_provincial.md) funds the apparatus while the return to them is positive and withdraws when it is not, letting the whole go by declining to hold it up.)

## Implementation

([The Retreat](place_the_retreat.md); the [shedding](process_shedding.md) process; the Provincial stops yielding the surplus once [the return](piece_the_return.md) to them has gone negative, so the [upkeep](piece_the_upkeep.md) goes unfunded and the society sheds the [layers](piece_the_layer.md) it can no longer pay for.)

## Targets

- [ ] the levy is weighed against what the apparatus returns to the base
- [ ] support continues while the return to the base is positive
- [ ] the surplus is withdrawn once the return to the base has gone negative
- [ ] the aggregate of many such withdrawals leaves the upkeep unfunded and the whole shed
