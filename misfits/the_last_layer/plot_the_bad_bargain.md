# Plot: The Bad Bargain

## Taxonomy

[The Last Layer](play_the_last_layer.md)

## Owner

- Project: khai-misfits

## Cue

[The Capital](place_the_capital.md), a new problem arriving past the peak of the curve.
A difficulty as real as any before, but the cheap solutions are long spent and only the dear ones remain.

## Action

[Diminishing](process_diminishing.md): the problem is met the only way the society knows, by adding another layer, though this one costs more than it yields.

## Stage

[The Successor](persona_the_successor.md), on [Carry the Whole](plan_carry_the_whole.md), adds another [layer](piece_the_layer.md) whose [return](piece_the_return.md) has already gone negative, because the method is proven and there is no other, and a problem left unmet is no answer at all.
The move is the same one that made the founder great, and on this stretch of the curve it makes the society poorer.

## Tension

Meeting the problem the proven way is the only method in hand, and the only method in hand now loses more than it saves.
There is no folly: the Successor applies the strategy that has always worked, and the alternative, to leave the problem unsolved, is no strategy at all, so adding the layer is the sole move available from the apex. But the return on it has fallen below its cost, so the plot closes with the problem met and the society weakened by meeting it, the same act rational for the founder and ruinous for the heir because of where each stands on the curve, and the method faithfully outliving the point where it paid.
