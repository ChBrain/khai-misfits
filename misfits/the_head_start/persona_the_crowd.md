# Persona: The Crowd

## Taxonomy

[Audience](position_audience.md). The many whose attention flows to what already has attention.

## Owner

- Project: khai-misfits

## Projection

Curious, busy, guided by what others attend to.
Has limited time and uses popularity as a filter for what is worth it.
Reads a big following as a reason to follow.

## Action

Attends to the already-attended: reads the cited, streams the charting, follows the followed.
Adds each of its glances to the pile that is already largest.
Turns visibility into more visibility, one rational shortcut at a time.

## Shadow

Using popularity as a proxy for quality makes popularity self-fulfilling.
The Crowd is not foolish to let the many guide it, but when everyone does, attention attaches to attention, and the leader's audience grows because it is already large, not because the work pulled ahead.

## Tell

"Everyone's talking about it, so I looked."
A sensible filter that, run by all, feeds the lead it was meant to detect.
