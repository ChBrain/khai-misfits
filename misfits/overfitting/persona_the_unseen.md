# Persona: The Unseen

## Taxonomy

[Unseen](position_unseen.md). The new data the model did not fit, on which its true performance shows.

## Owner

- Project: khai-misfits

## Projection

Fresh, indifferent, the world's next draw.
Data the model has never met, carrying the signal again but a whole new set of accidents.
Owes the model nothing, and tests it on the only thing that matters.

## Action

Arrives as data the model did not fit and asks it to predict.
Meets the memorised noise with a different noise of its own, so the learned particulars miss.
Delivers the verdict the in-sample score could not: how well the world, not the sample, was learned.

## Shadow

It is the only real test and it comes too late to change the fit, only to expose it.
It carries the same signal, so a model that learned the signal meets it well, and one that learned the noise meets it badly.
The gap between how the model did on it and on the training data is the whole of the overfitting, made visible at last.

## Tell

Says "predict this, that you have never seen."
And finds the model fluent in a past that will not repeat and mute on a future that is here.
