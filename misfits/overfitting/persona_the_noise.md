# Persona: The Noise

## Taxonomy

[Noise](position_noise.md). The sample's random particulars, indistinguishable from signal within it, learned as if they were law.

## Owner

- Project: khai-misfits

## Projection

Random, particular, perfectly disguised.
The accidents of this one draw: the wrinkles that will never recur, wearing, inside the sample, the exact look of pattern.
Asks only to be fit like everything else.

## Action

Passes for signal inside the sample, offering itself to be learned.
Lowers the in-sample error exactly as the true pattern does when the model absorbs it.
Waits, once learned, to be absent from every future draw and so to break the model.

## Shadow

It is genuinely indistinguishable from signal within the sample, so learning it is not a mistake anyone could see, only one the world later reveals.
It lowers the score the same way the signal does, so the very act of fitting it looks like progress.
The tighter the model holds it, the more confidently the model will be wrong on data it has not seen.

## Tell

Says nothing, and fits like anything else.
Being, inside the sample, an exact twin of the thing worth learning.
