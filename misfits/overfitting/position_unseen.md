# Position: Unseen

## Taxonomy

[Overfitting](play_overfitting.md). The seat of the new data the model did not fit, on which its true performance shows.

## Owner

- Project: khai-misfits

## Has

The world's next draw, never met by the model.
The same signal again, and a whole new set of accidents.
The only test that measures what was really learned.

## Orders

Break the model.
Ask it to predict data it did not fit.
Deliver the verdict the in-sample score could not.

## Loses

Nothing, and it comes too late to change the fit, only to expose it.
It carries the same signal, so a model that learned the signal meets it well and one that learned the noise meets it badly, and the gap between how the model does on it and on the training data is the whole of the overfitting.
The real test is the one that arrives only after the fit is fixed.

## Drives

Toward the true error,
on data the model never saw,
so the world's verdict lands after the score has already praised the model into the noise.
