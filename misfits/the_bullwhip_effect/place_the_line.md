# Place: The Line

## Taxonomy

[The Bullwhip Effect](play_the_bullwhip_effect.md). The supply line up which orders travel, each link ordering from the one above on the signal from the one below.

## Owner

- Project: khai-misfits

## Shown

A chain of links stretching up from the shop: retailer to wholesaler to distributor to plant, each passing an order to the next.
The order growing as it climbs, every stage adding its cover, the number arriving late and larger at each hand-off.

## Holds

The whole mechanism of the swing, stage by stage.
Here each link sees only its own customer and orders on that alone, so the line holds the chain of local decisions that turns a wobble into a wave, the place where cover compounds into distortion because no stage can see past the one below it.

## Offers

To each link, a clear view of its own customer's orders.
Because the line carries orders faithfully from stage to stage, it offers every link exactly the signal it acts on, the demand of the link beneath, so it hands each a real and usable number that is, unseen, already the swollen echo of the stage below.

## Withheld

The shopper, and the shape of the whole.
The line cannot show any link the true end demand or the swing building above and below it, because it carries only neighbour-to-neighbour orders, so it withholds the one view that would reveal the amplification, leaving each stage to mistake the chain's own caution for the market.
