# Plot: The Amplifying

## Taxonomy

[The Bullwhip Effect](play_the_bullwhip_effect.md)

## Owner

- Project: khai-misfits

## Cue

[The Line](place_the_line.md), the swollen order arriving as the only demand the next link sees.
Each stage reading its neighbour's cover as the market and adding a cover of its own.

## Action

[Amplifying](process_amplifying.md): [The Signal](persona_the_signal.md), holding [the signal's seat](position_signal.md), climbs from stage to stage growing as it goes, while [The Retailer](persona_the_retailer.md), holding [the retailer's seat](position_retailer.md), does at every link what the first one did.

## Stage

[The Signal](persona_the_signal.md), still on [Pass It Up](plan_pass_it_up.md), delivers each order upward stripped of its cause, so every stage takes the last stage's caution for demand.
[The Retailer](persona_the_retailer.md), holding [Cover the Order](plan_cover_the_order.md) at every link alike, covers the swollen figure it receives and passes something larger still, so the swing grows at each hand-off.

## Tension

The same prudence at every stage compounds into a wave.
There is no stage where anyone erred: each link only covered the demand it was shown, which is exactly what the link below did. But because that demand is already the stage below's cover, the caution stacks on caution, larger at every step, so the plot closes with a small wobble at the shop swollen, link by link, into a wild order climbing toward the top of the chain.
