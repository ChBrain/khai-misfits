# Place: The Name Tag

## Taxonomy

[Social Loafing](play_social_loafing.md). Where a contribution is made identifiable and tied to its hand.

## Owner

- Project: khai-misfits

## Shown

A part with a name on it, a small group, a task where each input can be seen.
The pool broken back into hands, each pull visible again.

## Holds

The restored effort.
Here what a hand does can be traced to the hand, so easing shows and pulling counts, and the effort returns because the being-seen returns with it.

## Offers

The one thing that lifts the loafing.
Identifiability, a group small enough that each matters, a task made to feel it matters, so no grip can quietly slacken unseen.

## Withheld

The easy warmth of the pool.
Naming each hand turns a team into scored individuals, so it lands as surveillance and a chill on the group feeling, and it holds only while it is held against the wish to pool the credit again.
