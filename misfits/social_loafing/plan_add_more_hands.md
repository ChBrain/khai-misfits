# Plan: Add More Hands

## Taxonomy

A **standing plan**: the intent of [the recruiter's seat](position_recruiter.md), inside [Social Loafing](play_social_loafing.md).

## Owner

- Owner: [Recruiter](position_recruiter.md)

## Direction

Answer a struggling task by growing the group. Add hands to the rope and expect the pull to grow with them, because many hands make light work. Meet a shortfall of output with a surplus of people.

## Orders

([The Piler](persona_the_piler.md) adds heads whenever the work is behind, sure the total will rise with the count.)

## Implementation

([The Group Task](place_the_group_task.md); the [diluting](process_diluting.md) process; [the ringelmann curve](piece_the_ringelmann_curve.md) drops the per-head pull as the group grows, widening [the process loss](piece_the_process_loss.md).)

## Targets

- [ ] more hands are added to the struggling task
- [ ] each added hand matters less and hides better
- [ ] the per-head pull falls as the group grows
- [ ] the total barely moves for all the added arms
