# Plan: Name the Contribution

## Taxonomy

A **standing plan**: the intent of [the reformer's seat](position_reformer.md), inside [Social Loafing](play_social_loafing.md).

## Owner

- Owner: [Reformer](position_reformer.md)

## Direction

Break the pool back into hands. Name the contribution, keep the group small, and make each input felt to matter, so the pull is seen and cannot quietly slacken. Trade some togetherness for some accountability.

## Orders

([The Namer](persona_the_namer.md) names the parts, shrinks the group, and ties the effort to the person.)

## Implementation

([The Name Tag](place_the_name_tag.md); the [naming](process_naming.md) process; [the social loafing](piece_the_social_loafing.md) lifts because effort made visible is effort restored.)

## Targets

- [ ] each contribution is made identifiable
- [ ] the group is kept small and the task made to matter
- [ ] the pull returns because it can be seen
- [ ] the naming holds, unless it is judged to spoil the team spirit
