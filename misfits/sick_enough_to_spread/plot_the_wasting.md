# Plot: The Wasting

## Taxonomy

[Sick Enough to Spread](play_sick_enough_to_spread.md)

## Owner

- Project: khai-misfits

## Cue

[The Reservoir](place_the_reservoir.md), the host bearing the optimum.
A parasite settled at the level that spreads best, multiplying in a host built from the same substance its copies are.

## Action

[Consuming](process_consuming.md): the germ spends the host to the degree the spread requires, drawing its copies from the host's own body, a harm worse than an achievable gentler state for the host.

## Stage

[The Carrier](persona_the_carrier.md) bears it from [the Vehicle](position_the_vehicle.md) on [Ride It Out](plan_ride_it_out.md), paying [the fare](piece_the_fare.md) of the spread out of itself, spent to the measure [the set point](piece_the_set_point.md) fixes, some of the damage buying the germ nothing at all.
The host is wasted, and there is no enemy in the room, only a thing that is spreading.

## Tension

The host is spent to the germ's exact need, and no one meant it any harm.
There is no malice: the germ takes nothing it is not built to take, drawing its copies from the host because there is nothing else to build them from, so the wasting is a fare and not an attack, set by what the spread costs and paid in the host's own substance. But the level is the one that spreads best and not the one the host would keep, and part of the harm is done in tissue the germ will never travel from, so the plot closes with a host consumed past any sense it would choose, bearing a sickness that answers to no will and appeals to none, the price of a passage it never booked and cannot lower by any effort of its own.
