# Persona: The Maker

## Taxonomy

[Maker](position_maker.md). The one who makes the shared artefact and circulates it to coordinate the communities, in good faith, and watches it seem to work. The archetype of the one who builds the boundary object and trusts the circulation.

## Owner

- Project: khai-misfits

## Projection

Thoughtful. Coordinating. Sure the object is joining the work.
Is the one who authors the roadmap, the spec, the metric, the prototype, the document the several groups will gather around, the one who reaches for a shared thing because the groups do not share a language.
No carelessness in it: the Maker builds the object well, plastic enough to be picked up by each community and robust enough to stay one object across the boundary, exactly the device the work needs.

## Action

Makes the shared artefact and circulates it between the communities, watching it pass cleanly from hand to hand and each group take it up and work from it.
Reaches for one common object rather than a shared vocabulary, because the object is what will let the groups coordinate without first agreeing on everything, and it does.
Does what a maker must: it builds a boundary object interpretively flexible enough to be shared, and takes the clean circulation for the coordination it was built to produce.

## Shadow

Not a naive builder, a competent maker of a real coordination device, and that is the whole difficulty: the object genuinely works, and its plasticity, the source of its power, is exactly what lets the meanings diverge under it.
Because the object passes cleanly and every community works from it, the Maker cannot see that the clean circulation is coordination on the object and not the meaning, so takes the shared artefact for a shared understanding and lets the object stand in for the conversation it was meant to open, trusting the circulation right up to the seam where the several readings meet and do not fit.
It builds a thing shareable across the boundary, and the shareability is exactly what hides that each community reads it otherwise.

## Tell

Says, in effect, "we are all working from the same thing," and means the object, not the meaning.
Says it watching the artefact circulate cleanly and every group work from it, and cannot tell the shared object from a shared understanding, so hears the clean circulation as coordination and trusts it to the seam.
