# Plan: Read It Home

## Taxonomy

A **standing plan**: the intent of [the community's seat](position_community.md), inside [The Common Object](play_the_common_object.md).

## Owner

- Owner: [Community](position_community.md)

## Direction

Take the shared object and read it into the terms of your own practice. Complete its face from the knowledge nearest your work, work from that reading as what the object plainly means, and assume the other communities read it the same, because the reading your practice supplies feels like what the object simply is.

## Orders

([The Community](persona_the_community.md) reads the object into its own local terms and takes the reading for the common meaning, then carries it home and builds it faithfully, one of several communities each working a different reading of one object.)

## Implementation

([The Home Ground](place_the_home_ground.md) where the object is read; the [homing](process_homing.md) process; [the object](piece_the_object.md) is completed into [the reading](piece_the_reading.md) nearest each practice, built toward [the seam](piece_the_seam.md) where the readings meet.)

## Targets

- [ ] the shared object is read into the terms of the community's own practice
- [ ] the competent local reading is taken for the object's plain meaning
- [ ] the other communities are assumed to have read the object the same way
- [ ] the reading is built faithfully on the home ground, one of several readings of one object
