# Position: Maker

## Taxonomy

[The Common Object](play_the_common_object.md). The seat that builds the shared artefact and circulates it to coordinate the communities, where the clean circulation is taken for the coordination it was built to produce.

## Owner

- Project: khai-misfits

## Has

A coordination problem and a device for it: several communities that do not share a language, and a shared object plastic enough to be read into each and robust enough to stay one thing.
A working device: the object is a real boundary object, competently built, that genuinely lets the groups work as one without first agreeing on everything.
No way to coordinate the groups without an object they can each read their own way, and no sign that the clean circulation is coordination on the object and not the meaning.

## Orders

Make the object and circulate it.
Build a shared artefact interpretively flexible enough to be picked up by each community, send it between them, and let the circulation join the work.
Trust the object, because the object is doing what it was built to do, passing cleanly and coordinating the groups.

## Loses

The chance of catching that the circulation coordinates the object and not the meaning, exactly because the object works.
Because the artefact passes cleanly and every community is working from it, the seat cannot see the shared thing as short of a shared understanding, so lets the object stand in for the conversation and trusts the circulation, one maker watching several communities read one object differently, and loses the divergence to the object's own success until the work is joined.
It gains a device that coordinates the groups, and loses the sight that the groups coordinate on the object alone.

## Drives

Toward the object circulating and the circulation taken for coordination,
the shared artefact felt as a shared understanding and trusted across the boundary,
so the seat that makes the object watches it join the work, sure to the seam that the meaning was joined too.
