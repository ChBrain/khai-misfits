# Plot: The Circulation

## Taxonomy

[The Common Object](play_the_common_object.md)

## Owner

- Project: khai-misfits

## Cue

[The Boundary](place_the_boundary.md), a coordination problem and a device for it: communities that do not share a language, and a shared object built to be read across them.
The Maker making the artefact plastic enough to cross into each world and sending it between the groups.

## Action

[Circulating](process_circulating.md): [The Maker](persona_the_maker.md), holding [the maker's seat](position_maker.md), sends the object across the boundary and watches each community take it up.

## Stage

The [maker](persona_the_maker.md), on [Circulate the Object](plan_circulate_the_object.md), sends [the object](piece_the_object.md) between the communities, its [plasticity](piece_the_plasticity.md) letting it pass cleanly into each practice and stay one thing across them.
The object circulates, every group works from it, and the Maker takes the clean passage for the coordination it was built to produce.

## Tension

Reaching for a shared object rather than a shared vocabulary is sound, and the soundness is what hides that the object coordinates the groups and not the meaning.
There is nothing to fault: the object is a real coordination device and the circulation is genuine. But a shared thing passing cleanly feels like joined work, so the plot closes with the object circulating and the Maker certain it is coordinating the communities, the shared face already carrying meanings it does not show.
