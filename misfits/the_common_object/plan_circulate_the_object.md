# Plan: Circulate the Object

## Taxonomy

A **standing plan**: the intent of [the maker's seat](position_maker.md), inside [The Common Object](play_the_common_object.md).

## Owner

- Owner: [Maker](position_maker.md)

## Direction

Build a shared artefact plastic enough to be read into each community and robust enough to stay one object, and send it between the groups. Let the circulation join the work, and trust the object, because it passes cleanly and every community is working from it, so the shared thing is doing what it was built to do.

## Orders

([The Maker](persona_the_maker.md) makes the boundary object and circulates it across the boundary, watching each community take it up, and takes the clean circulation for the coordination it was built to produce.)

## Implementation

([The Boundary](place_the_boundary.md) across which the object passes; the [circulating](process_circulating.md) process; [the object](piece_the_object.md) carries [the plasticity](piece_the_plasticity.md) that lets it be shared, its circulation felt as [the semblance](piece_the_semblance.md) of coordination.)

## Targets

- [ ] the shared artefact is built plastic enough to be read into each community and robust enough to stay one object
- [ ] the object is circulated across the boundary between the groups
- [ ] each community takes the object up and works from it
- [ ] the clean circulation is taken for the coordination the object was built to produce
