# Piece: The Plasticity

## Taxonomy

[The Common Object](play_the_common_object.md). The object's interpretive flexibility, the property by which it works by being read differently, and by which the meanings diverge unseen, the same trait carrying both.

## Owner

- Project: khai-misfits

## Place

Exercised across [the boundary](place_the_boundary.md), where it lets the one object arrive in each world.
Turned to divergence at [the home ground](place_the_home_ground.md), where it lets each community read the object its own way.

## Load Bearing

Without it the object determines its own single meaning, cannot be shared across communities that read differently, and there is no coordination and no divergence either.
With it the trap has its double edge: the plasticity that lets the object adapt to the local needs of each community while staying robust enough to hold a common identity is exactly the plasticity that lets each community read it into its own terms unseen (Star, on the concept's clarification, the interpretive flexibility that makes a boundary object work, and the caution that the flexibility is not the whole of it). The Plasticity is load bearing because it is one property doing two things: the source of the object's power to coordinate without consensus, and the source of the divergence it hides, so the piece carries the truth that the trait making the object shareable is the trait making the meanings part, the strength and the fault being the same thing.

## Apparent

A virtue, plainly: an object flexible enough to serve every community, adaptable, useful, the reason the shared thing works at all.
Seen at the surface, the Plasticity looks like pure strength, the object's happy capacity to fit every world, and the truth that the same flexibility that lets it be shared lets it be read differently, that the adaptability coordinating the groups is the adaptability dividing their meanings, stays hidden behind how plainly useful a flexible object is, until the readings meet.

## Yearbook

Built into [the object](piece_the_object.md) by [the maker](persona_the_maker.md), turned to divergence by each [community](persona_the_community.md) through [homing](process_homing.md).
It is the double edge of the trap: the Plasticity is what lets the Object be shared across [the boundary](place_the_boundary.md) and what lets each [reading](piece_the_reading.md) part unseen, the strength that makes [the semblance](piece_the_semblance.md) real and the flaw that grows [the divide](piece_the_divide.md).
