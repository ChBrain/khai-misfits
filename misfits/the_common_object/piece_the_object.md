# Piece: The Object

## Taxonomy

[The Common Object](play_the_common_object.md). The shared artefact itself, the document, the metric, the roadmap, the prototype, the spec that passes between the communities and coordinates them by being read into each.

## Owner

- Project: khai-misfits

## Place

Circulated across [the boundary](place_the_boundary.md), where it passes cleanly from world to world.
Read into each [home ground](place_the_home_ground.md), where its face is completed from the local practice.

## Load Bearing

Without it the communities have no shared thing to gather around and cannot coordinate without first agreeing on a vocabulary.
With it the trap has its device: a real boundary object, the artefact plastic enough to be picked up by each community and robust enough to stay one thing across them, so it coordinates the groups genuinely, without consensus (Star and Griesemer, on boundary objects, the shared form that inhabits several social worlds and satisfies the informational requirements of each). The Object is load bearing because it is the thing every community works from and each reads its own way: the common artefact that carries a different meaning in each set of hands, so the piece carries the shared face on which the divergence rides, coordinating the groups on the object while it splits them on the meaning.

## Apparent

A shared thing, plain and common: one document, one metric, one roadmap that everyone points to and works from.
Seen at the surface, the Object looks like exactly what it is, a real coordination device, and the truth that the one artefact is read differently in each community, that its single face carries several meanings, that the groups coordinate on the object and never on the sense, stays hidden behind how common a common thing looks, right up until the seam.

## Yearbook

Made and circulated by [the maker](persona_the_maker.md) through [circulating](process_circulating.md), read home by each [community](persona_the_community.md) through [homing](process_homing.md).
It is the device of the trap: the Object carries [the plasticity](piece_the_plasticity.md) that lets it be shared and read differently, completed into [the reading](piece_the_reading.md) nearest each practice, its circulation felt as [the semblance](piece_the_semblance.md), its several readings meeting at [the seam](piece_the_seam.md) to reveal [the divide](piece_the_divide.md).
