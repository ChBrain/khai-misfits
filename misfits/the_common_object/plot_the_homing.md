# Plot: The Homing

## Taxonomy

[The Common Object](play_the_common_object.md)

## Owner

- Project: khai-misfits

## Cue

[The Home Ground](place_the_home_ground.md), an object plastic enough to be read into each community's own practice.
Each Community taking the object to its own ground and reaching for the meaning nearest its own work.

## Action

[Homing](process_homing.md): [The Community](persona_the_community.md), holding [the community's seat](position_community.md), reads the object into its own terms and takes it for the object's meaning.

## Stage

Each [community](persona_the_community.md), on [Read It Home](plan_read_it_home.md), completes [the object](piece_the_object.md) into [the reading](piece_the_reading.md) nearest its own practice, one reading the roadmap as a sequence of builds, another as promises to customers, another as a statement of strategy.
Each takes its reading not for a reading but for what the object means, and assumes the other communities read it the same.

## Tension

Reading a plastic object into a working practice is necessary, and the necessity is what hides that the others read it otherwise.
There is nothing to fault: each reading is competent and the completion unavoidable. But the meaning a community's practice supplies feels like the object's own, so the plot closes with each community holding a different reading, every one certain it is the object's meaning and that the others hold it too.
