# Place: The Boundary

## Taxonomy

[The Common Object](play_the_common_object.md). The interface the object passes across between the communities, where the circulation is watched and taken for coordination and the meanings are not checked.

## Owner

- Project: khai-misfits

## Shown

The place the object crosses between communities, the interface where the engineers meet the clinicians, the sales meet the strategy, the shared artefact handed from one world to the next.
The Maker circulating the object, each Community taking it up across the boundary, and everyone watching the shared thing pass cleanly and reading the clean pass as the work joined.

## Holds

The clean circulation, and the reconciliation it spares.
Here the object crosses from world to world, so the Boundary holds the point where coordination is felt: a shared artefact plastic enough to pass into each practice and robust enough to arrive as one object, so it is where the presence of the common thing is taken for agreement on what it means, and where asking what the object means to each community comes to read as slowing a collaboration that is plainly moving, the groups joined by the object and the join untested.

## Offers

A device that crosses and communities that work from it.
Because the Boundary is where the object passes cleanly and each group takes it up, it offers a coordination reached and a collaboration felt, the artefact shared and the work moving as one, so it hands out the momentum of a shared thing, and hands it out on the object and not the meaning, the several local readings left unsurfaced beneath a single clean circulation.

## Withheld

Any sight of the several meanings under the one object.
The Boundary shows the object crossing but cannot show what each community takes it to mean, because the readings are made on the home ground and the circulation is public, so it withholds the divergence beneath the coordination, letting the presence of the common object stand for agreement on its sense, and, all clean circulation and no reconciled meaning, it withholds the difference between the readings until the home grounds build them and the seam brings them together.
