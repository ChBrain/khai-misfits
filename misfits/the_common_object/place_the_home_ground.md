# Place: The Home Ground

## Taxonomy

[The Common Object](play_the_common_object.md). Each community's own space where it reads the object into its own terms and builds its own reading, apart from the others and out of their sight.

## Owner

- Project: khai-misfits

## Shown

The place each community works, its own ground, the engineering bench, the clinic, the sales floor, the regulator's desk, where the shared object is read into the local practice and the local work is done.
Each Community completing the object's face from the knowledge nearest its own work, building faithfully to its own reading, apart from the other grounds and certain the others read the object the same.

## Holds

The competent local reading, and the divergence it grows.
Here each community reads the object into its own terms and builds, so the Home Ground holds the point where the several readings settle and take root: an object completed from the practice nearest each hand and worked faithfully into a body of work, so it is where the local reading is taken for the object's meaning, and where the difference from the other grounds is invisible because each ground is out of the others' sight, the reading competent and the divergence unseen.

## Offers

A competent body of work and a settled reading.
Because the Home Ground is where the object is read and built, it offers work done well and a meaning made concrete, the reading resolved and the ground moving forward on it, so it hands out the productivity of a shared object read into a practice, and hands it out on the local reading and not the common meaning, the several grounds each building a different reading of one object out of the others' sight.

## Withheld

Any sight of the other grounds' readings of the same object.
The Home Ground shows one community building its reading but cannot show what the others took the object to mean, because each ground is apart and the object is read locally, so it withholds the divergence between the grounds, letting each competent local reading stand as the object's meaning, and, all faithful local work and no reading compared, it withholds the difference between the readings until the separate work is carried to the seam.
