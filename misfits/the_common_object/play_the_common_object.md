# Play: The Common Object

## Estate

[khai-misfits](https://github.com/ChBrain/khai-misfits): the house that holds this production and answers for the whole run.

## Name

The Common Object.
Someone makes a thing to be shared, and it is a sensible thing to make: a spec, a roadmap, a metric, a prototype, a document that several groups can gather around, because the groups do not speak the same language and the object is what will let them work as one without first agreeing on everything. And it works. It genuinely works, which is the part that must be said first, because the object is plastic enough to be picked up by the engineers and the clinicians and the salespeople and the regulators, each reading it into the terms of their own practice, and robust enough that it stays recognisably the same object in every set of hands, so it holds an identity across the boundary while carrying a different sense within each community. That plasticity is not a flaw; it is the whole reason the object can be shared at all, the reason a museum's specimen or a firm's roadmap can coordinate people who would never converge on a common vocabulary. So the object circulates, and the circulation feels like coordination, because here is the thing about a shared artefact: it does not carry its meaning stapled to it, it carries a face each community completes from its own practice, and each community completes it correctly by its own lights. The engineers read the roadmap as a sequence of builds. The salespeople read the same roadmap as a set of promises they may make to customers. The executives read it as a statement of strategy. None misreads it; each reads it competently, into the meaning nearest to their own work, and because the object keeps passing between them, each takes the circulation for agreement, the shared thing for a shared understanding. And that is the fatal substitution: the object comes to stand in for the conversation it was meant to open, because once the artefact is in everyone's hands and everyone is working from it, to stop and ask "wait, what exactly does this mean to you" is to slow a collaboration that is visibly moving, to doubt a shared thing everyone can point to, to be the one who does not see how obviously it coordinates. So no one asks, because asking is costly and the coordination looks real, and each community carries the object home to its own ground and does its own work faithfully, exactly to the object as it reads it, and the object was one artefact and the work is several directions, and none of them is wrong, each is a competent use of the thing that was shared. And it holds, invisibly, right up until the separate work is brought together and does not fit, and only then, at the seam, does it come out that the one object everyone coordinated around meant a different thing in every community. Nobody misused it. Nobody was careless. The object was made in good faith and shared in good faith, each community read it competently by its own practice, and every group did faithfully what the object told it. It is only that an artefact plastic enough to cross a boundary is an artefact each community will read into its own terms, and a thing robust enough to stay one object holds its several meanings inside one shared face, so the coordination the groups felt was real was coordination on the object and never on the meaning, and the joined work everyone believed they were doing was the one thing they were not.

## Arc

The bend is this: a shared object is plastic enough to cross the boundary between communities and robust enough to stay one thing, so each community reads it correctly into its own practice, the circulation is taken for coordination, and the divergence of meaning stays hidden until the separate work is joined at the seam.
It runs on an artefact that genuinely coordinates by being read differently.
A boundary object works precisely because it is interpretively flexible: plastic enough to adapt to the local needs of each community that uses it, robust enough to maintain a common identity across them, so it can be shared without the communities first agreeing on what it means (Star and Griesemer, on boundary objects, coordination achieved across social worlds without consensus). So the several readings are not a fault but the mechanism, and this is what makes it a trap rather than an error, because the object is a real coordination device and its plasticity is the source of its power, not a defect in it.
And each community reads the object into its own terms. The engineers, the clinicians, the salespeople, the regulators each pick up the same artefact and complete it from the knowledge and practice local to their world, so the meaning each supplies is competent, specialised, and its own, the object made useful by being read the way that community reads (Carlile, on knowledge and boundaries, meaning transformed as it crosses the pragmatic boundary between functions). The reading is correct in every case, and correct differently in every case.
And the circulation is taken for coordination. Because the object keeps passing between the communities and each is working from it, the shared artefact comes to stand in for the shared understanding, the presence of the common thing mistaken for agreement on what it means, so the groups feel joined by the object precisely while they diverge under it (Bowker and Star, on classifications and standards as things that let communities cooperate without agreeing, coordination without consensus made durable and invisible). The object substitutes for the conversation, and the substitution feels like progress.
And the divergence runs unseen to the seam. Each community carries the object home to its own ground and does its own work faithfully, so one artefact becomes several competent bodies of work in different directions, and nothing compares the readings because the object looks shared and the work is done apart, until the separate pieces are brought together and do not fit (the divergence surfacing only at integration, when the joined work reveals the meanings were never one). The single object was several objects wearing one shared face, and only the seam shows it.
None of it is a misuse or a fault. The object was made in good faith and shared in good faith, each community read it competently by its own practice, and every group did faithfully what the object told it, so no one erred and the coordination was on the object and not the meaning. What makes the artefact shareable, that it is plastic enough to be read into each world, is exactly what leaves it open enough to be read differently, and a thing robust enough to stay one object is a thing that holds its several meanings inside one face (Star, on the concept's clarification, the boundary object working by tacking between the ill-structured shared form and the well-structured local uses, and failing only when the local uses are never reconciled).
Make the readings explicit and walk the meanings across before the work is joined: have each community say what the object means in its own concrete terms, surface the local reading and not just the shared form, and reconcile the meanings at the boundary rather than at the seam, and the divergence can be caught while the work is still separable: but this means slowing a collaboration that is visibly moving, doubting a shared thing everyone can point to, asking each community to spell out what the object plainly means to it, and treating a real coordination as suspect precisely because the object is doing so much of the work, so the meanings are reconciled only by paying, continuously, the cost of translation across the boundary the object was supposed to spare.

## Stakes

Whether several communities can share a meaning, when they can only share an object and the meaning each reads into it feels like the common one.
Every plot but the last moves the same way: toward the object circulating, which the Maker is right to share and each community right to read into its own practice, and away from the meaning reconciled, which would ask each community to spell out what the object plainly means to it and to doubt a coordination the object is visibly performing. The production cannot make the object carry one meaning, because an artefact plastic enough to cross the boundary is open enough to be read differently, and a thing robust enough to stay one object holds its several meanings inside one shared face, so the coordination the groups felt is a boundary object working as boundary objects do, coordination without consensus, not a misuse: no one erred and every reading was competent. It can only show that the object was made in good faith and shared in good faith and each community read it faithfully by its own practice, that the circulation coordinated the object and never the meaning, and that the one move that reconciles the meanings, making each reading explicit and walking it across the boundary, must be made by paying, continuously, the cost of the translation the object was supposed to spare, so the groups are coordinated and each is faithful and the joined work they believed they were doing is the one thing they were not.
