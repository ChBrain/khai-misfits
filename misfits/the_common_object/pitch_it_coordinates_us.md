# Pitch: It Coordinates Us

## Taxonomy

[The Common Object](play_the_common_object.md). The default key the whole production is read in.

## Owner

- Project: khai-misfits

## Tenor

Coordinated, productive, joined.
The plain good sense of several communities gathered around one shared artefact, each using it well by its own practice and certain the work is joined because the object keeps circulating between them.
Nothing false in it on its face: the object is a real coordination device, each community reads it competently, and the collaboration everyone feels is genuinely moving.

## Undertow

Under the coordination, an artefact plastic enough to be read into each community's own practice and robust enough to stay one object across the boundary, so every group reads it correctly and reads it differently.
Because the object keeps circulating, the shared thing stands in for a shared understanding, so the communities coordinate on the object and never on the meaning, and one artefact goes to each home ground as several, each built faithfully in a different direction, the gap invisible until the seam.
What makes the object shareable, that it is plastic enough to be read into each world, is what leaves it open enough to be read differently.

## Nerve

The feeling of working from the same object, the artefact plainly common and the collaboration visibly moving,
sure the work is joined, because the thing everyone points to is one thing,
while each community carries a different meaning home on the same shared face, and every home ground builds a faithful use of an object that meant several things.

## Echo

After the run: one shared object revealed at the seam to have meant a different thing in every community, each body of work faithful and none of them fitting,
no one having misused it and no one careless, only a plastic artefact read into each practice and a coordination that was only ever on the object,
and the one move that would have reconciled the meanings, making each reading explicit and walking it across the boundary, forgone because it slows a working collaboration and doubts a thing everyone can point to.
