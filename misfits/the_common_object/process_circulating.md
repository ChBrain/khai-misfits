# Process: Circulating

## Taxonomy

[The Common Object](play_the_common_object.md). The Maker sending the shared object between the communities, the clean passage across the boundary that coordinates the groups and is taken for coordination of the meaning.

## Owner

- Project: khai-misfits

## Initiated by

A coordination problem and a device for it: communities that do not share a language, and an object plastic enough to be shared across them.
At [the boundary](place_the_boundary.md), the Maker sending the artefact from world to world and each community taking it up.

## Direction

Toward the object shared and the sharing taken for coordination.
Because the communities cannot first agree on a vocabulary, circulating passes a shared artefact between them, and because the object is plastic enough to arrive in each practice and robust enough to stay one thing, each group takes it up and works from it, so the Maker takes the clean passage for the coordination it was built to produce, the shared object standing in for the shared understanding.

## Lever

The interpretive flexibility of the object.
Because the artefact is plastic enough to be read into each community and robust enough to stay one object across the boundary, circulating runs not on any forcing of agreement but on the object's genuine power to be shared without consensus, the coordination assembled from no coercion, only from a common thing each community can take up and read its own way.

## Echo

An object shared, and the sharing mistaken for the meaning joined.
The Maker has circulated a real coordination device, so what reads as the work joined is, in truth, coordination on the object alone, taken for agreement on its sense because the object passes cleanly, each community certain the others read it the same.
