# Process: Homing

## Taxonomy

[The Common Object](play_the_common_object.md). Each community reading the object into its own terms, the necessary completion that makes the artefact useful and is taken for the object's common meaning.

## Owner

- Project: khai-misfits

## Initiated by

A shared object that must be read into a practice to be used.
At [the home ground](place_the_home_ground.md), each community taking the artefact and completing its face from the knowledge closest to its own work.

## Direction

Toward the object read home and the reading taken for the meaning.
Because a shared artefact does not carry its own sense into every world, homing completes it with the reading nearest each practice, and because that reading is the one the community's knowledge supplies, it feels like what the object simply is, so each community takes its own resolution for the object's meaning and assumes the others read it the same, the local reading wearing the face of the common one.

## Lever

The competence of the local reading.
Because the meaning a community's practice supplies is specialised and correct by its own lights, homing runs not on any misreading but on the plain necessity of reading a plastic object into a working world, the divergence assembled from no error, only from each community completing the object with the sense nearest its own hand.

## Echo

An object read home, and the reading mistaken for the object.
The community has read the artefact competently, so what reads as shared understanding is, in truth, one reading among several, taken for the common meaning because it is competent and its own, each community certain the others read the object the same.
