# Plot: The Semblance

## Taxonomy

[The Common Object](play_the_common_object.md)

## Owner

- Project: khai-misfits

## Cue

[The Boundary](place_the_boundary.md), the object passing cleanly between the communities and each working from it.
The shared thing visibly common, the collaboration visibly moving, the check looking like doubt.

## Action

[Circulating](process_circulating.md): the object's clean passage stands in for the conversation, so [the maker](persona_the_maker.md) and every [community](persona_the_community.md) take the shared artefact for a shared understanding.

## Stage

Across [the boundary](place_the_boundary.md), the circulating [object](piece_the_object.md) gives [the semblance](piece_the_semblance.md) of coordination, the presence of the common thing taken for agreement on [the reading](piece_the_reading.md).
The groups feel joined because the object is visibly common, and to ask what the object means to each community reads as slowing a collaboration that is plainly moving, so no one reconciles the meaning.

## Tension

Trusting a shared object that is really coordinating is reasonable, and the reasonableness is what lets the object substitute for the conversation.
There is nothing to fault: the coordination on the object is genuine and the collaboration is really moving. But a visibly common thing feels like a shared meaning, so the plot closes with the object standing in for the conversation, the coordination believed and the one question that would test it made to look like doubt.
