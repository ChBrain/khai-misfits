# Process: Doubting

## Taxonomy

[The Stag Hunt](play_the_stag_hunt.md). The strategic uncertainty spreading through the party, each hunter unsure the others will hold.

## Owner

- Project: khai-misfits

## Initiated by

A prize that hangs on everyone and no way to see the others' resolve.
At [the hunt](place_the_hunt.md), each hunter glancing at the rest, weighing the stag's worth against a trust it cannot confirm.

## Direction

Toward the sure hare, by way of the fear the party will break.
Because the stag's whole value rests on all holding, and no hunter can be certain of all, doubting drives each to picture the line failing and itself left with nothing, so it turns an unconfirmable trust into a live risk, and makes the safe hare, which needs no trust, the guarded move.

## Lever

The gap between a prize that needs trust and one that does not.
Because the stag hangs on the others and the hare hangs on no one, any uncertainty falls entirely on the stag, so doubting works on that asymmetry alone, loading every fear onto the better prize and leaving the lesser one untouched, so the doubt itself does the choosing.

## Echo

A party each of whom now eyes the hare.
No one has peeled yet, but each has reason to, and each can see the others have the same reason, so doubting leaves the stag standing on a trust already thinning, the flinch not yet made and already rational for all.
