# Plot: The Doubt

## Taxonomy

[The Stag Hunt](play_the_stag_hunt.md)

## Owner

- Project: khai-misfits

## Cue

[The Hunt](place_the_hunt.md), each hunter looking to the others and unable to read their resolve.
The stag's worth hanging on all holding, and no way to be sure the rest will.

## Action

[Doubting](process_doubting.md): [The Partner](persona_the_partner.md), holding [the partner's seat](position_partner.md), wonders whether the hunter will stay, while [The Hunter](persona_the_hunter.md), holding [the hunter's seat](position_hunter.md), wonders the very same of the partner.

## Stage

[The Partner](persona_the_partner.md), weighing [Take the Sure Thing](plan_take_the_sure_thing.md), watches the hunter for a flinch and finds only a hunter watching back.
[The Hunter](persona_the_hunter.md), still on [Go for the Stag](plan_go_for_the_stag.md), holds for now but feels the trust thinning, each reading the other's caution as a warning about the line.

## Tension

Nothing has broken, and the prize is already unsafe.
There is no bad faith here: both hunters still prefer the stag, and neither has moved. But the stag's value rests on a trust neither can confirm, and each sees in the other the same uncertainty it feels, so the doubt loads entirely onto the better prize and leaves the sure hare untouched, and the plot closes with the line still whole and the safe hare, to every hunter, beginning to look like the wise move.
