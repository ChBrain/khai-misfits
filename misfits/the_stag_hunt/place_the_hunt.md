# Place: The Hunt

## Taxonomy

[The Stag Hunt](play_the_stag_hunt.md). The shared field where the party sets out, the choice between stag and hare before them and each other's nerve unknown.

## Owner

- Project: khai-misfits

## Shown

Hunters spread along a line, the stag ahead needing all of them and hares scattered in the grass within reach of each.
Every hunter glancing sidelong at the others, weighing the great shared prize against the sure small one, and no one able to see for certain what the rest have decided.

## Holds

The whole game in one field, both outcomes still open.
Here the party has not yet settled to stag or hare, so the hunt holds the fork itself, the moment before the choice tips, and holds with it the one thing that will decide it: not the payoffs, which everyone knows, but what each hunter believes the others will do.

## Offers

A great catch to a party that trusts itself, and a safe one to a party that does not.
Because the field carries both the stag and the hares, it offers each hunter the better prize if only the others hold and the sure prize if they might not, so it hands each the same fork and lets belief, not worth, decide which way they take it.

## Withheld

The others' minds.
The hunt cannot show any hunter what the rest have truly resolved, because each nerve is hidden until it acts, so it withholds the one fact that would make the stag safe, and leaves each to choose against a doubt the field itself will not lift.
