# Place: The Board

## Taxonomy

[Information Cascade](play_information_cascade.md). Where the choices are tallied, and only the choices.

## Owner

- Project: khai-misfits

## Shown

A running count of what everyone picked: the tally, the tote, the visible score.
A newcomer reading the numbers and weighing them against the one hint it holds.

## Holds

The inference, and the tipping point inside it.
Here the count of prior choices is read as a count of hints, so the board holds the moment the run outweighs a private signal and following becomes the right move.

## Offers

A clear, countable weight of evidence.
The board shows exactly how many went each way, so it offers an honest tally that any reasoner can use, and the more lopsided it is the more sensible it is to join.

## Withheld

The difference between a hint and an echo.
The board counts choices without marking which carried a signal and which merely copied, so a tally swollen by echoes reads exactly like one built on hints, and the newcomer, counting, cannot tell them apart.
