# Piece: The Tipping Point

## Taxonomy

[Information Cascade](play_information_cascade.md). The threshold where the run of observed actions outweighs a private hint.

## Owner

- Project: khai-misfits

## Place

At [the line](place_the_line.md), the place in the sequence where copying takes over.
Where one more concordant choice tips the balance against any single signal.

## Load Bearing

Without it deferring and signalling could go on trading places, and no cascade would lock.
With it there is a point of no return: once the run of revealed hints outweighs a single private one, every later actor is rationally bound to set their signal aside, so the tipping point is where informative choosing ends and copying begins, and it can be reached after only a few concordant actions. The Tipping Point is load bearing because it is the threshold that makes cascades start early and from little (Welch): a short run is enough to cross it, so the herd forms on a handful of signals and everyone after is right to defer.

## Apparent

The moment the answer becomes clear.
Crossing the threshold feels like the crowd converging on the truth, a consensus emerging, so the tipping point reads as the point where the matter is settled, when it is really the point where learning stops and echoing starts.

## Yearbook

Crossed along [the line](place_the_line.md), reached by [the inference](piece_the_inference.md) piling up.
It is the switch from signal to echo: past the Tipping Point every choice feeds [the cascade](piece_the_cascade.md) and none feeds the crowd's knowledge.
