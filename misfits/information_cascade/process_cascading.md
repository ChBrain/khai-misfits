# Process: Cascading

## Taxonomy

[Information Cascade](play_information_cascade.md). Each copy revealing nothing, so the run self-sustains and can flip on a breath.

## Owner

- Project: khai-misfits

## Initiated by

A tally large enough that following outweighs any one hint.
The tipping point crossed, and every newcomer arriving to a crowd it is right to join.

## Direction

Toward a run that feeds on itself and holds no new information.
Each copy is made on the crowd's signals, not the copier's, so it adds no hint, and the next reads one more echo and copies too, the herd extending on the strength of the first few.

## Lever

Copies that carry no signal, piling up as if they did.
Once following starts, actions stop revealing hints, so the crowd learns nothing further and rests on the handful of early signals, a base so thin that a single loud new action can outweigh it and turn the whole run around.

## Echo

A herd knowing far less than it holds, and easily reversed.
Private signals swallowed and unspoken, a certainty founded on almost nothing, and a cascade that can stampede the other way the moment a new signal lands.
