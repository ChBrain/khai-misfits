# Plot: The Inference

## Taxonomy

[Information Cascade](play_information_cascade.md)

## Owner

- Project: khai-misfits

## Cue

[The board](place_the_board.md), a run of the same choice tallied for all to see.
A newcomer weighs the count against the single hint it holds.

## Action

[Inferring](process_inferring.md): [The Follower](persona_the_follower.md), holding [the follower's seat](position_follower.md), reads the run as evidence of hints, while the choices of [The First](persona_the_first.md), holding [the opener's seat](position_opener.md), stand on the board as the run being read.

## Stage

[The Follower](persona_the_follower.md) tables [Follow the Crowd](plan_follow_the_crowd.md): weigh its one hint against the run, find itself outvoted, and copy, so [the inference](piece_the_inference.md) reads [the public action](piece_the_public_action.md) as signal up to [the tipping point](piece_the_tipping_point.md).
Past that point the run of revealed hints outweighs any single one, and deferring becomes the right move.

## Tension

The follower reasons perfectly, and its perfect reasoning is the first echo.
Reading actions as evidence is correct, and a long enough run really does outweigh one contrary hint, so deferring is sound, not weak. That is the turn: the copy it makes is chosen on the crowd's hints and not its own, so it carries no new signal, and the very correctness of the inference is what starts the crowd learning nothing from those who follow.
