# Piece: The Cascade

## Taxonomy

[Information Cascade](play_information_cascade.md). The self-sustaining herd: each copy revealing nothing, so information stops aggregating.

## Owner

- Project: khai-misfits

## Place

At [the line](place_the_line.md), the run extending on its own.
Where every new choice is an echo and none is a signal.

## Load Bearing

Without it the misfit would be a single bad choice, not a whole crowd's blindness.
With it the copying feeds itself: because each follower chooses on the crowd's information rather than its own, its action carries no hint, so the next learns nothing new and copies too, and the run rolls on resting entirely on the few signals before the tipping point. The Cascade is load bearing because it is the self-perpetuating herd the theory names (Bikhchandani, Hirshleifer & Welch; Banerjee), the state in which additional actors add numbers but no knowledge, so a whole crowd acts on almost none of what it collectively holds.

## Apparent

An overwhelming consensus.
A long unbroken run looks like a great weight of independent agreement, so the cascade reads as near-certainty, when its length is made of echoes and its certainty rests on the handful of hints that started it.

## Yearbook

Run by [the follower](persona_the_follower.md) and unbroken by [the dissenter](persona_the_dissenter.md), grown past [the tipping point](piece_the_tipping_point.md).
It is the herd that knows less than it holds: the Cascade silences [the private signal](piece_the_private_signal.md), and rests on so little that [the fragility](piece_the_fragility.md) can turn it.
