# Process: Inferring

## Taxonomy

[Information Cascade](play_information_cascade.md). Reading the run of actions as evidence of hints, until the run outweighs one's own.

## Owner

- Project: khai-misfits

## Initiated by

A newcomer with a private hint, and a run of prior choices to read.
The board showing a tally, and a mind weighing it against a single signal.

## Direction

Toward a private hint set aside for the crowd's.
The newcomer reads the actions as revealed hints, counts them against its one, and when they outweigh it, rationally defers and copies.

## Lever

The correct weight of many signals over one.
Reading actions as evidence is right, and a long enough run of them genuinely does outweigh a single contrary hint, so the inference that leads to copying is sound, not lazy, and following is the sensible move.

## Echo

A choice made on the crowd's hints, not the copier's.
The private signal outvoted and set down, the copy added to the board, and one more action that looks like evidence but carries no new hint.
