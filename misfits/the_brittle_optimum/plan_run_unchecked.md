# Plan: Run Unchecked

## Taxonomy

A **standing plan**: the intent of [the cascade's seat](position_cascade.md), inside [The Brittle Optimum](play_the_brittle_optimum.md).

## Owner

- Owner: [Cascade](position_cascade.md)

## Direction

Carry the first failure into the next part, and that into the next, following the tight couplings wherever they lead. Cross no buffer, because none is left, and take the whole system down in a chain that no single part can stop alone.

## Orders

([The Cascade](persona_the_cascade.md) runs the failure from part to part with nothing between them, meeting no give because the slack that would have damped it was cut.)

## Implementation

([The Break](place_the_break.md); the [shattering](process_shattering.md) process; [the coupling](piece_the_coupling.md) carries the failure onward into [the shatter](piece_the_shatter.md), the whole discharging at once what slack would have absorbed in pieces.)

## Targets

- [ ] the first failure is carried into the next part, and the next
- [ ] no buffer is crossed, because the give was trimmed away
- [ ] the failure follows the couplings across the whole system
- [ ] the chain runs as far as the tightness allows, which at the optimum is all the way
