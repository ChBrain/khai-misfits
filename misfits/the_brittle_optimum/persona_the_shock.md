# Persona: The Shock

## Taxonomy

[Shock](position_shock.md). The disturbance that eventually arrives, the jolt a looser system would have absorbed.

## Owner

- Project: khai-misfits

## Projection

Rare. Unscheduled. Certain in the long run.
Is the surge, the failure, the sudden demand, the outage: the kind of jolt that comes to any system if you wait.
No malice in it, and no timetable: only the near-certainty that something, someday, will land.

## Action

Arrives without warning and tests whatever give the system has left.
Strikes a part and asks the rest to absorb it, as a slacker system easily would.
Finds, in the lean optimum, no buffer to take the blow, and passes it straight on.

## Shadow

It is not the cause of the shatter, only its trigger: the fragility was built in before it came.
Because a system with slack would have flexed and a system without it breaks, the shock reveals a brittleness that was always there, so it is blamed as bad luck when it is merely the occasion, the match and not the tinder.
It was always going to come; only the give to meet it was optional.

## Tell

Says "here I am, as I always eventually am."
And finds the give that would have met it trimmed away for the numbers.
