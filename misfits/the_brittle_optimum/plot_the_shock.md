# Plot: The Shock

## Taxonomy

[The Brittle Optimum](play_the_brittle_optimum.md)

## Owner

- Project: khai-misfits

## Cue

[The Works](place_the_works.md) running lean at its optimum, and a disturbance arriving as one eventually does.
The sort of jolt a looser system would have taken without a mark, landing now on a taut one.

## Action

[Shattering](process_shattering.md): [The Shock](persona_the_shock.md), holding [the shock's seat](position_shock.md), strikes a part and asks the rest to absorb it, while [The Optimizer](persona_the_optimizer.md), holding [the optimizer's seat](position_optimizer.md), finds the give that would have met it long since cut.

## Stage

[The Shock](persona_the_shock.md), having tabled [Come Eventually](plan_come_eventually.md), arrives unannounced and tests whatever slack remains, which is none.
[The Optimizer](persona_the_optimizer.md), still on [Trim the Fat](plan_trim_the_fat.md), sees the lean system it built meet the jolt with nothing to spare, the margins that would have absorbed it gone for the numbers.

## Tension

The shock is only the trigger, and it finds a system with no give to meet it.
There is no fault in the shock's coming: something always comes eventually, and no one could have named which or when. But a system with slack would have flexed and this one has none, so the disturbance that a looser system would have shrugged off lands square on a taut one, and the plot closes with the jolt struck home and nothing between it and the rest of the system.
