# Position: Shock

## Taxonomy

[The Brittle Optimum](play_the_brittle_optimum.md). The seat of the disturbance that eventually arrives to test the system's give.

## Owner

- Project: khai-misfits

## Has

The certainty, over a long enough run, of a jolt: a surge, a failure, an outage, a sudden demand.
No schedule, and no obligation to announce itself.
The power only to test, never to build: it finds the give the system kept, or the give it cut.

## Orders

Come eventually.
Arrive unannounced, strike a part, and ask the rest to absorb it.
Test whatever slack remains, and expose whatever does not.

## Loses

Nothing, for it is only the trigger and never the cause.
Because a system with slack flexes and one without it shatters, the seat cannot itself decide the outcome, only reveal the brittleness already built, so it is blamed as bad luck when it is merely the occasion the fragility was waiting for.
It was always going to come, and only the give to meet it was ever in question.

## Drives

Toward a test that the calm deferred and could not cancel,
certain in the long run and unscheduled in the short,
so the disturbance is inevitable and only the give to absorb it is a choice.
