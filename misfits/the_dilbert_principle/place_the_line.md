# Place: The Line

## Taxonomy

[The Dilbert Principle](play_the_dilbert_principle.md). Where the work is done and the overmatched hand shows, the productive floor the shortfall costs and the manager watches.

## Owner

- Project: khai-misfits

## Shown

The seats where the actual work is made, the code and the ward and the shop, where doing it well or badly plainly tells.
The Duffer trying and falling short, the output suffering for the seat it cannot fill, and the Boss watching, weighing what to do.

## Holds

The failing and the choice, the honest shortfall and the exit the manager reaches for.
Here the work is close enough to judge, so the Line holds the point where incapacity is visible and must be answered: the shortfall is plain, dismissal is dear, and the soft exit is cheap, so it is where a blameless failure meets a manager's humane calculus and the kick upstairs is chosen.

## Offers

A place where competence can be seen for what it is.
Because the Line is where the work is done and doing it tells, it offers the one honest reading of ability the organisation has, so it hands out the plain evidence of who can do the work, and hands the overmatched, once read, straight toward the exit that lifts them out of sight of it.

## Withheld

A cheap way to keep competence where it is read.
The Line cannot make removing the overmatched as cheap as promoting them, so it withholds the low-cost honesty the manager would need to act on what the work shows, and, being the one place ability is plainly judged, it withholds that judgement from everywhere the worker goes next, for up is where the reading no longer reaches.
