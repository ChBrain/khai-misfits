# Plot: The Spiral

## Taxonomy

[First Do Harm](play_first_do_harm.md)

## Owner

- Project: khai-misfits

## Cue

[The Chart](place_the_chart.md), the new wound presenting as its own condition.
The harm arriving after the cure and wearing a fresh face, logged as a new complaint with no line drawn back.

## Action

[Treating](process_treating.md) again: [the healer](persona_the_healer.md), reading [the wound](piece_the_wound.md) as a fresh complaint, applies a second remedy that reaches past it and harms a third thing.

## Stage

Because [the alibi](piece_the_alibi.md) keeps the cure clear, [the healer](persona_the_healer.md), on [Treat the Complaint](plan_treat_the_complaint.md), treats the wound as its own condition, and [the remedy](piece_the_remedy.md) given again wounds again, link after link into [the cascade](piece_the_cascade.md).
Each intervention is genuine and each treats the last one's harm, the system filling with injuries no single cure will own.

## Tension

Treating the new wound is the same competent care as treating the first complaint, and it is what feeds the spiral.
There is nothing negligent: each wound presents as a real problem and each is diligently met. But the wound is the last cure's doing and the tie is never drawn, so the plot closes with the interventions chained, each genuine and each aimed at the harm the one before made, the cascade mounting and no link ever traced to the last.
