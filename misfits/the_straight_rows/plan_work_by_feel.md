# Plan: Work by Feel

## Taxonomy

## Owner

## Direction

Run the system as the ground actually is. Read the particular place by long practice: which soil, which season, which patch to leave, which custom holds, and keep the mixed stand and the rotation and the unplanned order that carry the whole. Hold the domain together by the thousand tacit adjustments no plan specifies, since the place is run by exactly the knowledge the survey cannot write down.
