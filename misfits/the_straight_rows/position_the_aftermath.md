# Position: The Aftermath

## Taxonomy

[The Straight Rows](play_the_straight_rows.md). The seat that inherits the legible system after the métis is cleared and meets the collapse the first yield concealed.

## Owner

- Project: khai-misfits

## Has

A legible system handed down, and a first success that vindicated it.
A plan to run faithfully: the plantation, the built scheme, the standard already imposed and already proven by the heavy first rotation.
No reserve left to draw on, because the métis that carried the first yield was cleared to make the domain legible, and nothing but the grid remains to run the place by.

## Orders

Run the map.
Tend the plantation and administer the plan exactly as designed, harvesting the second rotation as the first was harvested, keeping faith with a method the first success proved.
Work the grid as given, since the working knowledge that once ran alongside it is already gone.

## Loses

The domain itself, as the ground gives out beneath a plan that did everything right.
Because the first yield spent a métis the grid then cleared, the Aftermath inherits a system with no reserve, the soil unfed and the web gone, so the second rotation comes in sick and the third dead, the legible order failing on the very grid that made it governable.
It runs the plan perfectly and loses the place, the collapse set in motion a rotation before it arrived and past any faithful running to reverse.

## Drives

Toward a plan followed exactly,
and a ground that fails beneath it,
so the seat that inherits the legible order is the one the hollowing finds, blameless and unable to undo it.
