# Position: The Centre

## Taxonomy

[The Straight Rows](play_the_straight_rows.md). The seat that governs from a distance and can act only on what it can read, so it makes the domain legible and clears what it cannot hold.

## Owner

- Project: khai-misfits

## Has

A domain to govern and a distance to govern it across.
A hard precondition: a state cannot tax, conscript, plan, or feed a place it cannot see, so it must reduce the tangle to a grid it can read from the centre.
No way to hold a particular, unwritten knowledge in a table, and no acting on what cannot be recorded, so the seat can trust only what it can survey.

## Orders

Make it legible.
Impose the grid: replant the forest in rows, flatten the tenure into squares, straighten the quarter into a plan, and reduce the domain to a standard it can be read and run by.
See the place so it can be governed, since a centre cannot act on what it cannot hold.

## Loses

The métis, cleared with everything the grid cannot record.
Because the Centre keeps only what it can measure, the legible order it imposes razes the mixed stand and the customary practice and the unplanned order, the working knowledge that carried the domain, so the place is remade into the thin image the map can hold.
It gains a domain it can read and loses the knowledge that made the domain live, the survey mistaking what it cannot see for what is not there.

## Drives

Toward a domain remade to be read,
and a métis cleared to make it legible,
so the seat that must see to govern hollows the place in the very act of learning to run it.
