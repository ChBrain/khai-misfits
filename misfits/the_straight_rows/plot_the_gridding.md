# Plot: The Gridding

## Taxonomy

[The Straight Rows](play_the_straight_rows.md)

## Owner

- Project: khai-misfits

## Cue

[The Survey](place_the_survey.md), a domain the centre must govern and cannot read.
A planner with a distance to act across, and a tangle that must be reduced to a grid before it can be taxed, planned, or run.

## Action

[Simplifying](process_simplifying.md): the tangle is drawn as rows and squares and figures, and the ground is remade to match the map.

## Stage

[The Planner](persona_the_planner.md), on [Make It Legible](plan_make_it_legible.md), draws [the map](piece_the_map.md) and presses it onto [the forest](place_the_forest.md), remaking the domain into [the legible order](piece_the_legible_order.md) it can read, because a centre cannot act on what it cannot hold in a table.
Nothing is cruel in it, and the grid is a real tool: only the place is now what the map says it is.

## Tension

Reducing the domain to something the centre can read is the precondition of governing it, and the precondition is what remakes the place into a thin image.
There is no tyranny: a state must be able to see its domain to manage it, and the survey is the honest way to see. But the grid keeps only what it can measure, so the plot closes with the domain remade to be read and the ground pressed into rows, a legible order imposed that has already set aside everything it could not record.
