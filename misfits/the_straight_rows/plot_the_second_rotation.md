# Plot: The Second Rotation

## Taxonomy

[The Straight Rows](play_the_straight_rows.md)

## Owner

- Project: khai-misfits

## Cue

[The Forest](place_the_forest.md), the ground giving out where the first rotation stood heavy.
A successor running the proven plan faithfully, and a soil that has nothing left to give.

## Action

[Hollowing](process_hollowing.md): the system run on the grid alone loses the function the cleared mētis carried, and the stand comes in sick.

## Stage

[The Successor](persona_the_successor.md), on [Run the Map](plan_run_the_map.md), works [the legible order](piece_the_legible_order.md) exactly as designed with [the mētis](piece_the_metis.md) gone, and the heavy first [yield](piece_the_yield.md) gives way to [the blight](piece_the_blight.md), the second rotation sick and the worst of the third dead.
The plan was followed to the letter, and the letter had cleared its own ground.

## Tension

The plan is run perfectly, and running it perfectly is what meets the ruin the clearing set in motion.
There is no negligence: the Successor tends the plantation exactly as the vindicated design specifies. But the design cleared the mētis that fed the first yield, so the plot closes with the ground failing beneath a faithful hand, the legible order collapsing on the very grid that made it governable, a ruin no diligence can reverse and no tyrant ever authored.
