# Process: Control

## Taxonomy

[The Bounce](play_the_bounce.md). The reversal, where the story is spent to buy the truth.

## Owner

- Project: khai-misfits

## Initiated by

Someone holding a case back, untouched.
A hand withdrawn on purpose, so a comparison can exist.

## Direction

The counterfactual built, the trend judged.
The untreated run laid alongside the treated one.

## Lever

An untreated case set beside a treated one.
The difference read only across the pair, never within a single row.

## Echo

The bounce unmasked as reversion, at the cost of the flattering story.
A truth bought by giving up the credit.
