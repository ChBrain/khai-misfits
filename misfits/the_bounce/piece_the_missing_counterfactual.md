# Piece: The Missing Counterfactual

## Taxonomy

[The Bounce](play_the_bounce.md). What the result would have done untouched, unobserved and so unavailable.

## Owner

- Project: khai-misfits

## Place

At [the control](place_the_control.md), in the reading that is never taken because the action was.
The unseen comparison: the same trough left alone, the same peak not praised, the same team not shaken up. The result of doing nothing, which would show the bounce arriving all by itself, and which no one ever sees, because at the extreme, everyone acts.

## Load Bearing

Without it the intervention looks like the whole cause and the lesson is airtight.
With it the intervention shrinks to nothing, because the Missing Counterfactual is the only thing that could size the bounce: laid beside the treated result, the untreated one would rise almost as much, revealing the action's true effect as small or none. It is the load-bearing absence of the play, the measurement that would dissolve every false lesson, and it is missing not by accident but by structure, since acting on a case is exactly what destroys the chance to see it untouched.

## Apparent

Unnecessary, even absurd.
It looks like a pedant's demand: things were bad, we acted, they got better, and now someone wants to know what would have happened if we had sat on our hands? The counterfactual seems both obvious (surely the action helped) and impossible (you cannot un-act), so asking for it reads as either doubting the plain evidence or requesting the unavailable, and either way it is waved away.

## Yearbook

Demanded by [the doubter](persona_the_doubter.md), and supplied only when [the designer](persona_the_designer.md) holds a group back through [control](process_control.md).
It is the play's decisive absence: without the Missing Counterfactual, [the bounce](piece_the_bounce.md) cannot be told from an effect, and only building it back, by control or base rate or trend, ever exposes [the regression](piece_the_regression.md) beneath the lesson.
