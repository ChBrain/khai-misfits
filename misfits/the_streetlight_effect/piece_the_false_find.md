# Piece: The False Find

## Taxonomy

[The Streetlight Effect](play_the_streetlight_effect.md). The lit search's yield taken for the answer: the patch's contents, or its emptiness, reported as the truth.

## Owner

- Project: khai-misfits

## Place

At [the debrief](place_the_debrief.md), where the result is written up.
Where a finding drawn from the lit ground is presented as the finding.

## Load Bearing

Without it the wrong-ground search would end in an honest shrug, and no one would be misled.
With it the detour ends in a conclusion: the false find is whatever the lit patch held, taken for the answer, or whatever it lacked, taken for the thing's not existing, reported cleanly from a search that never entered the dark. The False Find is load bearing because it is where the harm lands (Freedman; the McNamara fallacy): it turns a search of the wrong ground into a confident claim about the truth, and because the search was thorough the claim is trusted, so a lit-patch answer stands as the answer and a lit-patch silence stands as proof there was nothing to find.

## Apparent

A settled result.
A clean finding from a careful search looks like the matter closed, so the false find reads as the answer or the all-clear, and the truth that it is a real result from precisely the ground the thing was not on, its confidence borrowed from the diligence that missed the mark, stays hidden behind how thorough the search appeared.
