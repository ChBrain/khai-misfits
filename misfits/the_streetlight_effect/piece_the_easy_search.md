# Piece: The Easy Search

## Taxonomy

[The Streetlight Effect](play_the_streetlight_effect.md). The cost gradient made act: effort flowing to the lit patch because searching there is cheap, wherever the thing is.

## Owner

- Project: khai-misfits

## Place

At [the streetlight](place_the_streetlight.md), where the effort collects.
In the pull that carries the search to the ground it can afford.

## Load Bearing

Without it the searcher might weigh where the thing is against where the looking is easy, and choose the thing.
With it the choice is made before it is felt: the easy search is the cost gradient in motion, the rational preference for a search that can be run over one that cannot, so effort flows to the light not by decision but by economy. The Easy Search is load bearing because it is the engine of the whole misfit (Simon): it is not a lapse but sound cost-reasoning, and it steers the search by the price of looking, which is exactly the variable that has nothing to do with where the answer lies.

## Apparent

Just being sensible about resources.
Directing the search to where it can actually be conducted looks like prudence, so the easy search reads as good economy, and the truth that it lets the cost of looking, not the location of the thing, decide where to look stays hidden behind how obviously wasteful the alternative would be.

## Yearbook

Run by [the searcher](persona_the_searcher.md) through [searching](process_searching.md), tabled as [Search Where It's Lit](plan_search_where_its_lit.md).
It is the cost gradient steering the look: the Easy Search flows to [the light](piece_the_light.md) and away from [the dark](piece_the_dark.md), dressed in [the thoroughness](piece_the_thoroughness.md) that makes its detour feel like progress.
