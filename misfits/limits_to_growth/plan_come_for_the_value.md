# Plan: Come for the Value

## Taxonomy

## Direction

Come in through the engine for the value it promises, and stay while the value is there. Take the service the capacity actually delivers, and when it thins past use, leave, without ceremony and without waiting to be counted out. Owe nothing to the total, only to the value you came for.
