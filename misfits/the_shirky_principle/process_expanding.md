# Process: Expanding

## Taxonomy

[The Shirky Principle](play_the_shirky_principle.md). The problem widened so the work never runs out.

## Owner

- Project: khai-misfits

## Initiated by

An original problem beginning, despite everything, to fade.
The threat of a shrinking mandate, and the reflex to grow rather than to finish.

## Direction

The remit widening, the threat redefined, new instances found before the old ones close.
The problem enlarged to fit the body, and each enlargement dressed as more mission.

## Lever

A budget that grows with the problem's footprint and shrinks when it closes.
So the body reaches for more problem the moment the current one recedes, because more problem is more resource.

## Echo

A finish line that moves out as fast as it is approached.
A mission that always turns out bigger than it looked, and a body that outgrows the very cause it was founded to end.
