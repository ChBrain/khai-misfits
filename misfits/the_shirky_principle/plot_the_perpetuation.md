# Plot: The Perpetuation

## Taxonomy

[The Shirky Principle](play_the_shirky_principle.md)

## Owner

- Project: khai-misfits

## Cue

[The frontier](place_the_frontier.md), the original problem starting, despite everything, to fade.
A shrinking problem is a shrinking budget, so its ending is felt as a threat, not a triumph.

## Action

[Expanding](process_expanding.md): [The Expander](persona_the_expander.md), holding [the empire-builder's seat](position_empire_builder.md), widens the mandate, finds new instances, and redefines the threat a size larger than before.

## Stage

[The Expander](persona_the_expander.md) tables [Widen the Remit](plan_widen_the_remit.md): grow the remit and the resources it commands, launch the next programme before the last problem ends, so [the perpetuation](piece_the_perpetuation.md) enlarges the problem to fit the body.
[The displacement](piece_the_displacement.md) lets the growth read as mission, so no one inside can tell the cause advancing from the institution feeding.

## Tension

The mission looks like it is advancing, and it is being manufactured.
Every widening is framed as more of the good work, and believed, so the body grows in perfect sincerity. But the growth arrives precisely when the original problem recedes, because a fading cause is a falling budget, so the institution enlarges the problem to match itself, and calls the manufacture service. The finish line moves out exactly as fast as it is approached.
