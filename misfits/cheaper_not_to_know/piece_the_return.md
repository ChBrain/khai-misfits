# Piece: The Return

## Taxonomy

[Cheaper Not to Know](play_cheaper_not_to_know.md). The expected benefit of informing: the value of a better outcome, times the near-zero pivot, so small it never covers the cost.

## Owner

- Project: khai-misfits

## Place

Computed at [the Booth](place_the_booth.md) through [Weighing](process_weighing.md), the sum the Voter checks against the Cost before the ballot is ever marked.
Not a place so much as the answer a citizen reaches when the value of a better outcome is multiplied against the chance of bringing it about: the number that closes the reckoning, one way or the other.

## Load Bearing

The Return is load bearing because it is the calculus itself: the expected benefit of informing is near zero because it is realized only if one's vote is decisive, which it almost never is (Downs). State the arithmetic precisely: near-zero, the Pivot, times a large stake, the value of a better outcome, is still near-zero, so the size of what is at stake in the election does not rescue the sum. This is exactly why informing is a losing investment for the individual however valuable the knowledge would be collectively, and it is the reason the Cost, real and positive as it is, never gets covered. Guard against overclaiming a precise figure for the Return; what is defensible is that it is near zero, not that it is exactly nought or exactly one in some stated number for every case.

## Apparent

Indifference to the stakes. Not caring how the election turns out.
It looks like a citizen who has simply decided the outcome does not matter to them, discounting the value of a better result to nothing. That reading confuses the size of the stake with the size of the Return: the stake can be enormous and the Return still near zero, because it is the Pivot doing the multiplying, not any indifference to what is at stake.
