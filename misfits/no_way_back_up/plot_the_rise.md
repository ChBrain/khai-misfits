# Plot: The Rise

## Taxonomy

[No Way Back Up](play_no_way_back_up.md)

## Owner

- Project: khai-misfits

## Cue

[The Colony](place_the_colony.md), the least-loaded class gone and no way to remake it.
A population whose cleanest class has just been lost, with no reversal coming and no mixing done.

## Action

[Loading](process_loading.md): the fewest-mutations number steps up by one, the floor rising to the next class and fixing there.

## Stage

[The Toss](persona_the_toss.md)'s loss raises [the rising floor](piece_the_rising_floor.md) a notch, [the back-mutation](piece_the_back_mutation.md) that would lower it never arriving, [the Copier](persona_the_copier.md) carrying the new minimum faithfully forward.
The floor moved once, in one direction, and closed the way back behind it.

## Tension

The minimum climbs a notch it can never climb back down, and nothing did it but the sum of blameless acts.
There is no agent of the rise: the class was lost to chance, the reversal that would undo it is too rare to come, and the copying that carries the rest forward is faithful. But with every route down shut, the loss of the cleanest class is not a dip but a step, so the plot closes with the floor risen and fixed, the population's best permanently one mutation heavier, and the class that is now cleanest already the rarest and already one unlucky draw from the next rise, the climb set to repeat with nothing in the lineage able to reverse a single notch of it.
