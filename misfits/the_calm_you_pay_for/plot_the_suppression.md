## Action

[The Disturbance](persona_the_disturbance.md), on [Press In Band](plan_press_in_band.md), presses its signal squarely in the working band, honest about where it lives, giving the designer a true target to answer.

## Tension

There is nothing to fault here: the right move, done well.
The plot closes with [the calm](piece_the_bought_calm.md) bought in the band that matters, [the sensitivity](piece_the_sensitivity.md) pushed below one, the loop smooth exactly where anyone would look. What the same tuning did elsewhere is not yet on stage.
