# Place: The Bench

## Taxonomy

[Prevalence-Induced Concept Change](play_prevalence_induced_concept_change.md). Where the cases are judged one by one, each read in the light of the ones lately seen.

## Owner

- Project: khai-misfits

## Shown

The seat of judgement, the screen or the desk or the panel where cases arrive one after another to be called.
The assessor weighing each against the run behind it, flagging the thing and passing the fine, feeling the same steady eye throughout.

## Holds

The judging, and the reference it runs on.
Here each case meets not a written rule but the memory of the recent ones, so the Bench holds the point where a fair call becomes a moving one: the standard the assessor applies is the recent company itself, so it is where honest judgement is quietly hostage to whatever has lately come across the desk.

## Offers

A fair reading of each case, one at a time.
Because the Bench judges every case honestly against what it knows, it offers exactly the even-handedness the task asks, so it hands out a consistent-feeling verdict on each, and hands it out calibrated to a company that will drift, so the evenness is real and the anchor is not.

## Withheld

A standard that does not move with the room.
The Bench cannot judge against a mark it has not got, so it withholds the absolute rule the assessor believes it is using, giving it only the drifting reference of the recent run, and, offering no vantage above the cases, it withholds any sight of the drift, so the eye stays sure it is fixed while the ground slides beneath it.
