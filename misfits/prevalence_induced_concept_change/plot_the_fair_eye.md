# Plot: The Fair Eye

## Taxonomy

[Prevalence-Induced Concept Change](play_prevalence_induced_concept_change.md)

## Owner

- Project: khai-misfits

## Cue

[The Bench](place_the_bench.md), an assessor set to find the thing and a run of cases to judge.
The true cases common, the eye calibrated to them, each new case read in the light of the last.

## Action

[Judging](process_judging.md): [The Assessor](persona_the_assessor.md), holding [the assessor's seat](position_assessor.md), calls each case in the light of the recent ones, feeling one unchanging standard.

## Stage

[The Assessor](persona_the_assessor.md), on [Judge by the Light](plan_judge_by_the_light.md), weighs each case against the recent run, flags the thing where it sees it and lets the fine case pass.
The line of the concept sits where the common true cases put it, and the eye is sure that line is fixed.

## Tension

The fairest way to judge is the way that will betray it.
There is nothing to fault: reading each case against recent company is sound perception, efficient and honest, and the assessor feels perfectly consistent. But the standard it trusts as absolute is pinned to the company it keeps, so the plot closes with the eye judging fairly and well, its line quietly hostage to a run of cases that is about to change.
