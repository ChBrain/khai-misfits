# Plot: The Mirage

## Taxonomy

[Prevalence-Induced Concept Change](play_prevalence_induced_concept_change.md)

## Owner

- Project: khai-misfits

## Cue

[The Ledger](place_the_ledger.md), the tally holding level though the true cases fell.
Each vanished true case replaced by a recategorised mild one, the count refusing to drop.

## Action

[Creeping](process_creeping.md): [The Seeker](persona_the_seeker.md), holding [the seeker's seat](position_seeker.md), reads the flat count as a problem undefeated and drives the effort on, while [The Assessor](persona_the_assessor.md), holding [the assessor's seat](position_assessor.md), keeps feeding it cases from a widened line.

## Stage

[The Seeker](persona_the_seeker.md), on [Root It Out](plan_root_it_out.md), takes the level tally as proof the danger stands and presses the work into ever milder ground, not letting up.
[The Assessor](persona_the_assessor.md), still on [Judge by the Light](plan_judge_by_the_light.md), keeps flagging the thing at the drifted line, so the count the seeker trusts never falls.

## Tension

The effort chases a boundary that moves as it is approached.
There is no villain: the seeker is conscientious and trusts its numbers, and the assessor judges in good faith, each doing its honest part. But the count is flat only because the concept crept, so the plot closes with the work driven hardest at the mildest cases, the definition ballooning as the reality shrinks, and the win nowhere on the page.
