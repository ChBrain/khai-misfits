# Position: Assessor

## Taxonomy

[Prevalence-Induced Concept Change](play_prevalence_induced_concept_change.md). The seat that judges by recent company, fair for one case and creeping for all.

## Owner

- Project: khai-misfits

## Has

A run of cases to judge, a task to find the thing, and an eye that reads each case against the recent ones.
A sound instinct: judge by context, the efficient way perception works, and it feels like one fixed standard.
No absolute rule to hold to, only the drifting reference of what it has lately seen.

## Orders

Judge by the light.
Read each case honestly against the run you know, flag the thing where you see it, and let the fine case pass.
Hold, as it feels to you, to one unchanging standard.

## Loses

The fixed meaning of its own standard, exactly by judging fairly.
Because it reads each case against recent company and the company mildens as the true cases fall, the seat cannot both judge by context and keep its line still, so the fairness that feels constant is what walks the boundary outward, and it cannot feel the drift because from the inside the standard never moves.
It stays as strict as ever and swallows ever milder cases, and has no vantage from which to tell the two apart.

## Drives

Toward a line set by recent company,
fair for each case and widening across the run,
so the honest eye that would find the thing is the eye that keeps redefining it.
