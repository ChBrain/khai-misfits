# Plot: The Priming

## Taxonomy

[The Good Move](play_the_good_move.md)

## Owner

- Project: khai-misfits

## Cue

[The Beaten Path](place_the_beaten_path.md), a run of problems all yielding to one method.
A series of successes, each solved the same way, the groove deepening with every problem the method clears.

## Action

[Priming](process_priming.md): the repeated success installs the set, teaching the mind not just the method but the readiness to reach for it before the next problem is read.

## Stage

[The Master](persona_the_master.md) solves problem after problem down [the Beaten Path](place_the_beaten_path.md), on [Take the Good Move](plan_take_the_good_move.md), and each correct answer lays [the set](piece_the_set.md) a little deeper, the disposition built by the very practice that makes them expert.
Nothing here is a mistake, and that is the point: the learning is real, the method sound, the readiness earned.

## Tension

The set being installed is the honest fruit of practice, and it is the exact thing that will blind.
There is no error in the run: every problem is solved correctly, and the mind that learns to reach for a method that has never failed is learning the right lesson. But the same readiness that makes the next familiar problem quick is the readiness that will fire before a search on the problem where a better move waits, so the plot closes with the solver more expert and more set at once, the disposition laid down sound and ready and already carrying, unseen, the block it will become.
