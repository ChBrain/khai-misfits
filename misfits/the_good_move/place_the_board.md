# Place: The Board

## Taxonomy

[The Good Move](play_the_good_move.md). The problem laid out whole, where the good answer and the better one are both present, both legal, both there to be found on the same field.

## Owner

- Project: khai-misfits

## Shown

A problem and a solution, cleanly solved.
Shown as competence rewarded: the familiar answer applied, the case closed, a correct move made on a board that plainly admitted it, everything the surface offers saying the problem is done.

## Holds

Holds two answers at once, the found one and the better one, on the same field.
The Board is where the whole problem sits open, so the good move and the better move are both present on it together, both available from the first, the difference between them not that one is there and one is not but only which one the looking reaches.

## Offers

Offers a true problem with a real solution, and a better solution alongside it.
Here the good answer is genuinely correct and the board genuinely admits it, so the Board offers sound work done well, and in the same layout it offers the better move too, present the whole time, so the field that rewards the good answer is the field that was holding a better one all along.

## Withheld

Withholds that the better move is on the same board.
What the Board hides is not the better answer's existence but its location in the looking: it sits on the field in plain legality, withheld only in that the good answer, found first, sends the attention elsewhere, so the problem that looks fully solved is the problem still holding, unlooked-at, the move that would have solved it better.
