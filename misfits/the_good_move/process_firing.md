# Process: Firing

## Taxonomy

[The Good Move](play_the_good_move.md). The set going off, the primed solution presenting itself at once when the new problem matches the pattern, an answer in hand before any search begins.

## Owner

- Project: khai-misfits

## Initiated by

[The Master](persona_the_master.md), at [the Board](place_the_board.md), meeting a problem that matches the kind the set was primed on.
Begins the instant the new case is recognised as one of the known kind, the match alone enough to set the readiness off without a decision to apply it.

## Direction

Turns a match into a ready answer.
Because the problem fits the primed pattern, the familiar solution rises unbidden and fast, so firing converts the recognition of a kind into a solution already in hand, the good answer produced before the search that would weigh it against any other has even started (Luchins).

## Lever

The speed of recognition.
Because seeing the kind and reaching the answer are nearly one motion, firing works through how quickly the match delivers, and its quickness is the whole value of the set: an answer that arrives this fast is what expertise feels like from the inside, so the lever is exactly the readiness that practice earned.

## Echo

A first answer, held before the looking.
The echo of firing is a correct solution already in the mind before any search begins, so whatever looking follows starts from a position that is filled rather than open, the good answer occupying the ground the search would otherwise have crossed.
