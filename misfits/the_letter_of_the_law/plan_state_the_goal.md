# Plan: State the Goal

## Taxonomy

A **standing plan**: the intent of [the specifier's seat](position_specifier.md), inside [The Letter of the Law](play_the_letter_of_the_law.md).

## Owner

## Direction

Turn the intent into a measurable objective the optimiser can score. Choose the proxy that best stands in for the goal, write it as well as it can be written, and hand the search a target. Accept that the number must carry a goal the number is not, because there is no complete objective to write and something must be given to maximise.
