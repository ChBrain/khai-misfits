# Piece: The Thicket

## Taxonomy

[Vetocracy](play_vetocracy.md). The proliferation of veto points, each added for a good reason, summing to a maze.

## Owner

- Project: khai-misfits

## Place

At [the chamber](place_the_chamber.md), grown dense over time.
In the accreted gates, each earned by some past reason, no proposal can clear.

## Load Bearing

Without it a single veto would be a check, not a paralysis.
With it the checks become a wall: the thicket is the accumulation of veto points, each added to guard against some real failure, that together leave no path a proposal can take through them all. The Thicket is load bearing because it is where legitimate parts make an illegitimate whole (Olson on the accretion of veto-holding interests; Rauch's demosclerosis): no one built the maze, it grew one reasonable check at a time, so it cannot be blamed on any hand and cannot be cleared by removing any villain.

## Apparent

A mature system's full set of safeguards.
A dense array of checks looks like thoroughness, a system that has learned from every past mistake and posted a guard at each, so the thicket reads as accumulated wisdom, and the truth that the same accumulation is what no proposal can pass, that maturity and paralysis are the same growth, stays hidden behind how prudent each added check was.

## Yearbook

Grown from every [veto](piece_the_veto.md) through [vetoing](process_vetoing.md), the maze [Cut the Thicket](plan_cut_the_thicket.md) would clear.
It is the sum no hand built: the Thicket turns a set of legitimate checks into an impassable whole, the structure that, priced against action, makes [the asymmetry](piece_the_asymmetry.md) bite.
