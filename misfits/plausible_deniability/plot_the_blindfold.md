# Plot: The Blindfold

## Taxonomy

[Plausible Deniability](play_plausible_deniability.md)

## Owner

- Project: khai-misfits

## Cue

[The office](place_the_office.md), where knowledge is a liability the principal must not hold.
The deed is in motion below, and the one thing the principal now needs is to not be told of it.

## Action

[Blinding](process_blinding.md): [The Principal](persona_the_principal.md), holding [the principal's seat](position_principal.md), declines to be told and does not ask, while [The Broker](persona_the_broker.md), holding [the broker's seat](position_broker.md), filters the chain upward to keep it clean.

## Stage

[The Principal](persona_the_principal.md) tables [Let It Fall](plan_let_it_fall.md) in its blinding aspect: structure the reporting so the doing never arrives, so [the blindfold](piece_the_blindfold.md) makes the ignorance real.
[The Broker](persona_the_broker.md) tables [Pass It Down](plan_pass_it_down.md), keeping [the cut-out](piece_the_cut_out.md) clean, so nothing the agent knows can climb to the top.

## Tension

The principal works hardest at not knowing the one thing it most wants done.
There is no lie here, only an arrangement: the questions unasked, the reports unread, the knowledge kept out on purpose, so that the ignorance is genuine. That is the shield forming: a not-knowing built so carefully that the denial it will support is not a falsehood at all, a true ignorance manufactured to make a true acquittal.
