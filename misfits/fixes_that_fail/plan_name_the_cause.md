# Plan: Name the Cause

## Taxonomy

A **standing plan**: the intent of [the diagnostician's seat](position_diagnostician.md), inside [Fixes that Fail](play_fixes_that_fail.md).

## Owner

- Owner: [Diagnostician](position_diagnostician.md)

## Direction

Follow the loop past the relief to its delayed effect, and trace the worse return through the gap back to the fix. Say the unwelcome thing: the remedy everyone trusts is the cause. Warn that more of it will mean more trouble, and expect not to be believed.

## Orders

([The Diagnostician](persona_the_diagnostician.md) traces the worsening to the fix and is waved off, because the fix so plainly works.)

## Implementation

([The Morning After](place_the_morning_after.md); the [backfiring](process_backfiring.md) it reads; [the delay](piece_the_delay.md) hides the link, so [the backfire](piece_the_backfire.md) is laid to chance and not to the fix that caused it.)

## Targets

- [ ] the worse return is traced through the delay to the fix
- [ ] the fix is named as the cause, against everyone's trust in it
- [ ] the warning that more fix means more trouble is given
- [ ] the true diagnosis is dismissed beside a symptom the fix just cured
