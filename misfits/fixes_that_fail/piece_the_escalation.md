# Piece: The Escalation

## Taxonomy

[Fixes that Fail](play_fixes_that_fail.md). The reinforcing response: more of the same fix applied as the worse return reads as too little fix.

## Owner

- Project: khai-misfits

## Place

At [the treadmill](place_the_treadmill.md), the fix reapplied faster and larger.
Where each worse return draws more fix and each more fix deepens the return.

## Load Bearing

Without it a single backfire would be a lesson, and the fix would be dropped.
With it the backfire is misread as a shortfall, so the response to the worsening is more of the thing causing it, and the loop tightens instead of teaching, the reach for the fix worn into reflex and the trouble compounding. The Escalation is load bearing because it is the reinforcing loop closing (Kim), the point where a slow trap becomes a fast one, and the reason a fix that fails is not abandoned but doubled, driven hardest exactly when it is doing the most harm.

## Apparent

A vigorous, responsible response.
Applying more of the fix as the problem worsens looks like taking the trouble seriously, so the escalation reads as diligence, and the mounting effort passes for control while the ground goes backward.

## Yearbook

Driven by [the fixer](persona_the_fixer.md) through [escalating](process_escalating.md), grown from [the backfire](piece_the_backfire.md) misread.
It is the loop at speed: the Escalation compounds the trouble and hands the debt to [the heir](persona_the_heir.md), each cycle costing more and buying less.
