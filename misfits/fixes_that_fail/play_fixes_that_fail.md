# Play: Fixes that Fail

## Estate

[khai-misfits](https://github.com/ChBrain/khai-misfits): the house that holds this production and answers for the whole run.

## Name

Fixes that Fail.
A problem flares and a fix is applied that works, at once and unmistakably ... but the fix has a second effect, slow to arrive and hard to trace, that feeds the very problem it relieved, so the trouble returns, worse, and is met with more of the same fix, which relieves it again and worsens it again, faster, until the cure and the disease are the same act and no one suspects it, because the fix so plainly works.

## Arc

The bend is this: the fix is the cause, and its success is what hides it.
It runs on two effects of one act, on different clocks.
The first effect is relief: the fix eases the symptom now, visibly, reliably, so it earns trust the instant it is applied. The second effect is a side effect: the same fix, through a longer loop, feeds the problem, and this arrives late and tangled with everything else, so it is never laid at the fix's door. Between the fast relief and the slow backfire sits a delay, and in that gap the fix looks like pure remedy.
None of it is folly. The fixer is right that the fix works, the one it relieves is right to want it, and when the problem returns worse the natural reading is not that the fix caused it but that there was not enough fix, so more is applied, and the loop that was turning slowly begins to spin. The relief shortens, the backfire deepens, and the reach for the fix becomes reflex, each dose buying less and costing more, the trouble compounding under a remedy no one will question because it so obviously helps.
Let the fix go, bear the symptom long enough for the loop to unwind, and address the cause the fix was hiding, and the problem can finally fall: but the fix really did work, letting it go means choosing the pain now for a gain that comes late and unpromised, and the fix is right there.

## Triggers

**[The Fix](plot_the_fix.md)**
A problem flares and demands an answer now. The Fixer applies a fix that works at once, and the Relieved sees the symptom go and credits the fix. Closes when the relief is real and immediate, and the fix is trusted for it.

**[The Backfire](plot_the_backfire.md)**
Cue: after a delay, the fix's second effect arrives and feeds the problem. The Heir inherits a symptom returned worse, and the Diagnostician sees that the fix, not chance, is what deepened it. Closes when the trouble is worse than before and the cause is the cure.

**[The Escalation](plot_the_escalation.md)**
Cue: the problem is back and worse, and the fix worked before. More of the same fix is applied, the loop spins faster, and the Diagnostician's warning that the fix is the cause is waved off as absurd. Closes when the reach for the fix is reflex and the compounding is underway.

**[The Letting Go](plot_the_letting_go.md)**
Cue: someone moves to stop the fix, bear the symptom while the loop unwinds, and address the cause it was hiding. The Breaker forgoes the relief and holds through the worse-before-better. Closes when the loop is broken and the cause is reached ... or the symptom flares too hard, the fix is reached for again, and the cycle resumes.

## Stakes

Whether a problem is ever traced to its cause, or only ever met with the thing that is feeding it.
Every plot but the last moves the same way: toward the fix, which relieves the symptom now, and away from the cause, which the relief keeps hidden. The production cannot make a delayed side effect announce itself, so the fix looks like pure remedy right up to the point where the trouble it causes is unmistakable, by which time the reach for it is habit. It can only show that no one was reckless, that the fix truly worked and the worse return truly looked like too little fix, and that the one thing that would break the loop, letting the fix go and bearing the symptom while the cause is faced, asks a certain pain now for an uncertain relief later, against a fix that is proven and immediate and right there.
