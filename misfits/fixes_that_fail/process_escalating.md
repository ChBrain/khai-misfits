# Process: Escalating

## Taxonomy

[Fixes that Fail](play_fixes_that_fail.md). More of the same fix applied to a worse return, spinning the loop faster.

## Owner

- Project: khai-misfits

## Initiated by

A problem returned worse, and a fix that worked before.
The reading, natural and wrong, that a deeper trouble means too little of the remedy.

## Direction

Toward a tightening loop and a compounding problem.
More fix relieves the symptom a little and feeds it a lot, so the return is worse still, drawing more fix again, each cycle shorter and steeper than the last.

## Lever

A worse return read as too little fix.
Because the fix's relief is real and its harm is hidden, the sensible response to the worsening is more of what causes it, so the loop is driven hardest exactly when it is doing the most damage.

## Echo

A reflex, a debt, and a problem out of hand.
The reach for the fix worn automatic, each dose buying less and costing more, and the compounded trouble handed on to whoever inherits it.
