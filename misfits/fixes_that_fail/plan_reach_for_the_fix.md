# Plan: Reach for the Fix

## Taxonomy

A **standing plan**: the intent of [the fixer's seat](position_fixer.md), inside [Fixes that Fail](play_fixes_that_fail.md).

## Owner

- Owner: [Fixer](position_fixer.md)

## Direction

Meet the problem with the fix that works. Apply it, watch the symptom go, and move on. When the trouble returns, apply it again, and when it returns worse, apply more, because a remedy this reliable is plainly the answer and the worse return can only mean too little of it.

## Orders

([The Fixer](persona_the_fixer.md) applies the effective fix, credits the relief, and escalates when the trouble deepens.)

## Implementation

([The Emergency](place_the_emergency.md); the [fixing](process_fixing.md) process; [the fix](piece_the_fix.md) eases the symptom now and earns its trust through [the relief](piece_the_relief.md), the visible result that hides the loop.)

## Targets

- [ ] the problem is met with the fix and the symptom goes at once
- [ ] the immediate relief is read as proof the fix is the answer
- [ ] the fix is reached for again each time the trouble returns
- [ ] the worse return draws more fix, the reflex quickening each cycle
