# Position: Fixer

## Taxonomy

[Fixes that Fail](play_fixes_that_fail.md). The seat that applies the fix, and applies it again.

## Owner

- Project: khai-misfits

## Has

A fix that eases the symptom fast and reliably.
The trust that comes of a result everyone can see.
No sight of the second effect, which arrives too late and too tangled to trace.

## Orders

Meet the problem with the fix.
When it returns, apply the fix again.
When it returns worse, apply more of the fix.

## Loses

Nothing it can see, and the problem it is quietly deepening.
Its remedy relieves now and feeds the trouble later, so the worse return reads as too little fix and draws more, and the seat escalates the very cause it is trying to cure.
Right about the relief every single time, and wrong about the loop.

## Drives

Toward the fix, faster with each cycle,
away from any cause the relief keeps hidden,
so a proven remedy becomes a reflex that compounds the trouble it answers.
