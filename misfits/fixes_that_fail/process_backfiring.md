# Process: Backfiring

## Taxonomy

[Fixes that Fail](play_fixes_that_fail.md). The fix's second effect arriving late, feeding the problem it relieved.

## Owner

- Project: khai-misfits

## Initiated by

A fix applied, and a longer loop it also set in motion.
The same act that relieved the symptom, working now through a slower path back to the trouble.

## Direction

Toward a problem deeper than before, and hard to trace.
The relief fades, the second effect lands, and the symptom returns worse, at a distance in time that hides where it came from.

## Lever

A delay between the act and its harm.
The gap severs the effect from its cause, so the backfire is never laid at the fix's door, and the worsening reads as fresh trouble or bad luck rather than the remedy's own return.

## Echo

A worse problem and an unsuspected cause.
The trouble deepened by the very fix that eased it, the link buried in the delay, and the stage set to read the worse return as a call for more fix.
