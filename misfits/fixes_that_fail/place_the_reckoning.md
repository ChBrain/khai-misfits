# Place: The Reckoning

## Taxonomy

[Fixes that Fail](play_fixes_that_fail.md). Where the fix is let go and the cause is finally faced.

## Owner

- Project: khai-misfits

## Shown

The fix set down, the symptom flaring unanswered, and the cause it hid brought into the open.
A hard, quiet stretch where the relief is refused on purpose and the loop is left to unwind.

## Holds

The worse-before-better, and the chance under it.
Here the symptom is loud and unhandled while the loop falls quiet, and only in that unrelieved space can the cause be reached, so the place holds both the pain of the cure and the cure itself.

## Offers

The one real exit from the loop.
Stopping the fix is the sole move that lets the compounding unwind and the cause be worked, so the reckoning offers the only way the problem actually falls.

## Withheld

Comfort, and any promise it will pay.
The symptom is not relieved, the cure is slow, the result is late and unpromised, and the fix is right there the whole time, so the reckoning withholds every reassurance, and holds only if the pain is borne on faith.
