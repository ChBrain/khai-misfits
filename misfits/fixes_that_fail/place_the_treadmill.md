# Place: The Treadmill

## Taxonomy

[Fixes that Fail](play_fixes_that_fail.md). Where more of the fix is applied, faster, and the loop compounds.

## Owner

- Project: khai-misfits

## Shown

The fix reapplied at a quickening pace, each dose larger, each interval shorter.
Great effort spent, and the trouble not falling but deepening, the running faster and the ground going backward.

## Holds

The escalation, in full motion.
Here the worse return draws more fix and the more fix deepens the worse return, so the place holds the loop at speed, the reach for the remedy worn into reflex and the compounding underway.

## Offers

The feeling of vigorous response.
Everyone is working hard, the fix is being applied more than ever, and the sheer activity reads as control, so the treadmill offers the comfort of effort in place of the fact of progress.

## Withheld

A way off that isn't stopping.
The only exit is to step down and let the belt run out, which means bearing the flare unrelieved, so the treadmill withholds any escape that keeps the fix, and running harder only spins it faster.
