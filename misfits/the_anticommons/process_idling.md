# Process: Idling

## Taxonomy

[The Anticommons](play_the_anticommons.md). The resource sitting unused, its value bleeding away while the owners hold.

## Owner

- Project: khai-misfits

## Initiated by

A deal that could not close and a resource left where it stood.
The last yes never given, the assembler gone, the pieces still scattered and still held.

## Direction

Toward a worth that decays in place, unused.
The shop stays dark, the drug unshipped, the block unbuilt, and the value that use would have created is simply never made, quarter after quarter.

## Lever

Underuse where a commons would have overuse.
The same fragmentation that would have starved a common field of restraint here starves the resource of consent, so it is wasted not by being taken but by never being touched.

## Echo

A resource wanted by all and moved by none.
The need met poorly at the margins, the owners no richer for their pieces, and a value that existed only in a use no one could unlock.
