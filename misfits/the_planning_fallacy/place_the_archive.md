# Place: The Archive

## Taxonomy

[The Planning Fallacy](play_the_planning_fallacy.md). Where the record of tasks already done is kept, the base rate available and unread.

## Owner

- Project: khai-misfits

## Shown

The tasks that came before and how long each really took, estimate against actual, line after line.
Every line running the same way, the actual past the estimate, so the distribution is plain to anyone who opens it: tasks like this take longer than they are ever thought to.

## Holds

The outside view, kept and complete.
Here sits the reference class the new estimate belongs to, the correction the board never carried, so the archive holds the one number that would price the task right, drawn not from its story but from the count of its kind, and holds it whether or not anyone comes to read it.

## Offers

The base rate, to any estimate willing to be checked against it.
Because the archive records what happened and not what was planned, it offers the distribution the inside view ignores, the overruns totalled and one-directional, so it offers the estimate its correction in advance, needing only that the number be counted into the class it plainly joins.

## Withheld

The particulars, and the will to consult it.
The archive holds averages, so it withholds what is genuinely special about this task, which is the very reason it is waved off as not applicable, and holding no claim on attention it withholds itself from the board by default, read only when someone thinks to leave the plan and go looking for the record.
