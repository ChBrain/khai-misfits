# Plot: The Plan

## Taxonomy

[The Planning Fallacy](play_the_planning_fallacy.md)

## Owner

- Project: khai-misfits

## Cue

[The Drawing Board](place_the_drawing_board.md), a task laid out step by step and each part given about its time.
The plan drawn clean along the run where the work goes well, and the sum of it becoming a date that feels honestly reached.

## Action

[Estimating](process_estimating.md): [The Planner](persona_the_planner.md), holding [the planner's seat](position_planner.md), builds the date from the plan, while [The Sponsor](persona_the_sponsor.md), holding [the sponsor's seat](position_sponsor.md), waits for a number low enough to approve.

## Stage

[The Planner](persona_the_planner.md), having tabled [Plan the Work](plan_plan_the_work.md), sums the parts along the smooth run and reads the care taken as reason to trust the date.
[The Sponsor](persona_the_sponsor.md) tables [Set the Date](plan_set_the_date.md), rewards the low estimate, and presses the promise toward the number that clears the gate.

## Tension

The most careful estimate is the one set most surely too soon.
There is no haste to find: the task is genuinely thought through, the plan is real work, the date is honestly meant. But a plan can only draw the run where the work goes well, so the number built from it is the best case, and the sponsor's want of a low bid shades it lower still, so the estimate leaves the board trusted in exact proportion to the diligence that made it optimistic, a promise sincere and unmeetable at once.
