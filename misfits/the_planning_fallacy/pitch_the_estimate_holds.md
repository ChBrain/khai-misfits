# Pitch: The Estimate Holds

## Taxonomy

[The Planning Fallacy](play_the_planning_fallacy.md). The default key the whole production is read in.

## Owner

- Project: khai-misfits

## Tenor

Reasonable, diligent, thought through.
The steadiness of a task laid out step by step, each part given about the time it should take, and a date that feels honestly arrived at.
Nothing hasty in it: a plan is made, the sum is taken, and the estimate holds.

## Undertow

Under the diligence, an estimate built from the one run where nothing goes wrong.
The plan traces the path where the work goes well, so the number it yields is the best case wearing the face of the likely one, and the record that would price it right is set aside as not this task.
The date is not met by mistake: it was never a date the world was going to keep.

## Nerve

The feeling of promising a date you sincerely believe,
because you have thought the task through in detail and the detail is real,
while the detail is exactly the plan's smooth run, and the surprises that always come have no place in it to be counted.

## Echo

After the run: a task delivered late and over, like the one before it and the one before that,
a plan that was honest and an estimate that was meant and a record that was there the whole time unread,
and the same gap as last time, waiting to be called bad luck again.
