# Plan: Plan the Work

## Taxonomy

A **standing plan**: the intent of [the planner's seat](position_planner.md), inside [The Planning Fallacy](play_the_planning_fallacy.md).

## Owner

- Owner: [Planner](position_planner.md)

## Direction

Lay the task out step by step and estimate it from the plan. Sum the parts at about the time each should take, reason from the detail of this case, and reach a date thought through and honestly meant. Trust the plan as the forecast, for a task thought through in detail is a task well estimated.

## Orders

([The Planner](persona_the_planner.md) builds the date from the plan, summing the parts along the run where the work goes well, and reads the care taken as reason to trust the number.)

## Implementation

([The Drawing Board](place_the_drawing_board.md); the [estimating](process_estimating.md) process; [the plan](piece_the_plan.md) drawn along the smooth run yields [the estimate](piece_the_estimate.md), built from the inside view and blind to the base rate.)

## Targets

- [ ] the task is laid out step by step and each part given about the time it should take
- [ ] the estimate is reasoned from this task's own detail, the run where nothing in particular goes wrong
- [ ] the plan thought through is read as an estimate to trust
- [ ] the date is reached and promised, built from the inside view alone
