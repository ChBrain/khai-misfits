# Plan: Escalate

## Taxonomy

A **standing plan**: the intent of [the spiral's seat](position_spiral.md), inside [Accidental Adversaries](play_accidental_adversaries.md).

## Owner

- Owner: [Spiral](position_spiral.md)

## Direction

Turn every wound into a blow and every blow into a wound. Take each defensive move as fuel for the next, make each round larger than the one before, and let both parties keep supplying the grievances. Convert the partnership into a rivalry, one justified move at a time.

## Orders

([The Spiral](persona_the_spiral.md) compounds the exchanges, running on reasonable moves alone, so the loop that drags both down needs no villain, only two parties each defending itself.)

## Implementation

([The Feud](place_the_feud.md); the [spiralling](process_spiralling.md) process; [the spiral](piece_the_spiral.md) compounds each round and hides its own shape, since each party sees only its own half, the harm it takes and not the harm it deals.)

## Targets

- [ ] each wound is turned into a blow and each blow into a wound
- [ ] every round is made larger than the harm it answers
- [ ] the loop runs on reasonable moves and needs no malice
- [ ] its shape stays hidden, each party seeing only the harm it receives
