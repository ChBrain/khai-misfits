# Piece: The Spiral

## Taxonomy

[Accidental Adversaries](play_accidental_adversaries.md). The reinforcing loop of mutual undermining, compounding allies into adversaries.

## Owner

- Project: khai-misfits

## Place

At [the feud](place_the_feud.md), where it runs.
In the rounds of blow answering blow, each larger than the last.

## Load Bearing

Without it a single misread injury would be a quarrel, not a ruin.
With it the quarrel becomes a slide: the spiral is the reinforcing loop in which each retaliation deals fresh unseen harm, misread again as attack and answered again, so the exchanges compound and the partnership tips, round by round, into rivalry. The Spiral is load bearing because it is what makes the misfit a system and not an incident (the reinforcing archetype): it needs no villain, only two parties each defending itself, and it hides its own shape because each sees only its own half, so the loop that drags both down is built entirely of moves each was sure were right.

## Apparent

A rivalry with a clear aggressor, if only you ask the right party.
An escalating feud looks like proof that one side is hostile, so the spiral reads as a conflict with a culprit, and the truth that it is a self-feeding loop with no author, that both parties are right about their wounds and wrong about their cause, stays hidden behind how obviously, to each side, the other started it.

## Yearbook

Run by [the spiral](persona_the_spiral.md) through [spiralling](process_spiralling.md), tabled as [Escalate](plan_escalate.md).
It is the loop that turns allies to adversaries: the Spiral compounds each [misattribution](piece_the_misattribution.md) into the next, dragging down [the symbiosis](piece_the_symbiosis.md) it was built on, broken only by [See the Whole](plan_see_the_whole.md).
