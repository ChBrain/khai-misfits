# Plan: Write the Win

## Taxonomy

A **standing plan**: the intent of [the Byline](position_the_byline.md), inside [The File Drawer](play_the_file_drawer.md).

## Owner

- Owner: [The Byline](position_the_byline.md)

## Direction

Spend the write-up where it will be read. Draft and submit the significant result, and file the null, because a finding earns a citation and a career and a non-finding earns a rejection and a shrug, so the scarce months of writing go to the result the record will keep. Answer the return the literature offers, since nothing in the reward makes a sound null worth the labour of writing it up.

## Orders

([The Author](persona_the_author.md) writes up the significant and drawers the null, spending effort where it pays and losing the record its tempering evidence at the source.)

## Implementation

([The Bench](place_the_bench.md); the [selecting](process_selecting.md) process at its first fall; the Author submits [the significant](piece_the_significant.md) and files [the null](piece_the_null.md), so [the record](piece_the_record.md) is filtered before any reviewer sees it.)

## Targets

- [ ] the significant result is written up and submitted, the null filed in the drawer
- [ ] the choice is arithmetic, not fraud: a finding pays and a non-finding does not
- [ ] the record is filtered at the bench by the one best placed to know the null was sound
- [ ] the tempering evidence never leaves the drawer, and the literature reads surer than the work
