# Position: The Check

## Taxonomy

[The File Drawer](play_the_file_drawer.md). The replicator's seat, where a published finding is tested again and the failed check is drawered by the same filter that kept the original.

## Owner

- Project: khai-misfits

## Has

A doubt, and a replication run carefully and at power to settle it.
The mechanism a science trusts to catch its false positives: run the study again, and see whether the effect holds.
No way to make a failed replication into a finding, since a null is a null whether it is a first result or a check.

## Orders

Try to correct.
Replicate the published study, and if the effect is not there, take the null to the record as the correction it is.
Run the check the science depends on, since a finding unchecked is a finding untested.

## Loses

The correction, to the same drawer, at the same door.
Because a failed replication is a null and nulls do not print, the check that would correct the record is refused by the record for the very reason it matters, that it found nothing, so the false positive stands in the literature while the challenge sits in a drawer.
It runs the check soundly and loses it to the filter, the correction screened by the screen it came to undo.

## Drives

Toward a false positive left standing,
and its correction filed unseen,
so the seat that carries the science's self-correction cannot correct through a filter that keeps only the positive.
