# Plan: Try to Correct

## Taxonomy

## Direction

Run the study again and settle the doubt. Replicate the published finding carefully and at power, and if the effect is not there, take the null to the record as the correction it is, because a finding unchecked is a finding untested and self-correction is what a science is for. Carry the check to the door, since a false positive stands until something overturns it.
