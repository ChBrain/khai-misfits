# Position: The Byline

## Taxonomy

[The File Drawer](play_the_file_drawer.md). The author's seat, where a finding is written up and a null is filed, and the record is filtered before any reviewer sees it.

## Owner

- Project: khai-misfits

## Has

A finding and a null, from equally sound work, and a scarce clock to write with.
A plain arithmetic: a finding earns a citation and a career and a non-finding earns neither, so the months of writing up have one place to go.
No way to make a null pay and no reason a rejection will not follow it, so the sound non-finding is the one the seat sets aside.

## Orders

Write the win.
Draft and submit the significant result, and file the null, spending the write-up where it will be read and letting the non-finding rest unwritten.
Answer the return the record offers, since nothing in the reward makes a null worth the labour.

## Loses

The null, to the drawer, and with it the record's tempering evidence.
Because the Byline submits the finding and shelves the non-finding, the literature is filtered at the source by the person best placed to know the null was sound, so the honest result that would have pulled the estimate down never leaves the bench.
It keeps the win and loses the null, and the record is edited before it is ever reviewed.

## Drives

Toward a record filtered at the bench,
and a drawer filling with sound nulls,
so the seat that faked nothing still makes the literature surer than its own results were.
