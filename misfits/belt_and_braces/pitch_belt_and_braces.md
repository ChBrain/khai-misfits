# Pitch: Belt and Braces

## Taxonomy

[Belt and Braces](play_belt_and_braces.md). The default key the whole production is read in.

## Owner

- Project: khai-misfits

## Tenor

Prudent, defended, sound.
The plain good sense of a system with a spare behind every guard, the reliability figure climbing with each backup added, more redundancy reading as more safety and every addition warranted.
Nothing false in it on its face: the backup was added in good faith, the arithmetic was done correctly, and each spare removes a real single point.

## Undertow

Under the defence in depth, a backup wired and cooled and crewed and trusted with the rest, so it is not the independent part the arithmetic assumed.
Because each addition adds coupling the designers did not draw, shares a common cause that fells the redundant set at once, and diffuses the vigilance it was meant to add, past a point each further backup lowers the true reliability while the figure that counts it still climbs.
What makes a backup read as safety, that it adds a chance in parallel, is what the coupled system denies it.

## Nerve

The feeling of leaving a system safer for the spare, the figure higher and the single point removed,
sure the redundancy raised the reliability, because another guard behind the guard could only add,
while each backup couples the parts a little more, shares one more cause, relaxes one more watch, and the true reliability, uncounted, has already begun to turn.

## Echo

After the run: a redundant set felled together by the one thing that was never redundant, the many guards shown to have been one, the belt and the braces down of a single shared cause,
no one having been reckless and no one careless, only a backup added in good faith into a coupled system, and a figure that climbed while the reliability fell,
and the one move that would have held it, counting the true independence before adding, forgone because it refuses a spare that reads as safe and doubts a figure that climbs.
