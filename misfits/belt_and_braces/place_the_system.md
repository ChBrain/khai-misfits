# Place: The System

## Taxonomy

[Belt and Braces](play_belt_and_braces.md). Where the backups sit wired and cooled and crewed together, and where the interactions grow and the shared cause waits, unseen, until it fells the redundant set at once.

## Owner

- Project: khai-misfits

## Shown

The place the backups live after the design, the running plant, the fielded system, the operation where each spare sits inside the whole, wired to the same bus and watched by the same crew.
The Backups standing ready, the Operators running the system trusting them, and the parts interacting along paths the diagram did not draw.

## Holds

The coupling, and the common cause waiting at it.
Here each backup sits inside the system and not apart from it, so the System holds the point where redundancy becomes coupling: more parts and more paths between them, more spares and more cause shared among them, so it is where the complexity grows in the open and unseen at once, every backup ready and every one reachable by the single cause that reaches them all, the independence the design assumed absent in the wiring.

## Offers

Defence in depth in place, spares standing ready on every side.
Because the System is where the backups sit and the operators run, it offers real cover in place and a margin really there, the redundant set ready and the operation moving, so it hands out the reserve of many guards, and hands it out coupled, each guard sharing a cause with the next and each watch shared until it is dropped, the many chances wired into one.

## Withheld

Any sign, until the cause arrives, that the many are one.
The System holds each ready backup but cannot show it against the shared cause, because the coupling is latent and the common mode has not yet fired, so it withholds the collapse until the one cause comes, letting every spare stand ready as an independent chance, and, all defence in depth and no independence, it withholds the moment the belt and the braces go together until the flood or the power or the premise they share reaches them all at a stroke.
