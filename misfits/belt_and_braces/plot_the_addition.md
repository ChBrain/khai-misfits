## Action

## Stage

[The Designer](persona_the_designer.md), on [Add Another](plan_add_another.md), places [the backup](piece_the_backup.md) behind the primary and reads [the reliability](piece_the_reliability.md) figure climbing with the spare.
The single point is removed and the number rises, and the coupling the backup brings and the cause it shares go untraced, unseen beneath a gap plainly closed.

## Tension

Adding a spare to a real single point in good faith is not recklessness, and the good faith is what keeps the designer from seeing the coupling.
There is nothing careless: the gap is real and the arithmetic is sound. But a backup wired and cooled and crewed with the rest is not the independent chance the number counts, so the plot closes with the spare added and the figure climbing, the designer sure the system is safer and the turn it begins unseen.
