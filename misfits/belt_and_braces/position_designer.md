# Position: Designer

## Taxonomy

[Belt and Braces](play_belt_and_braces.md). The seat that adds the backup to remove a single point, where a real gap closed is felt as a system made safer.

## Owner

- Project: khai-misfits

## Has

A real single point of failure and a backup to remove it: the gap the primary might leave and the spare that covers it.
A sound arithmetic: for parts that fail independently, a parallel backup raises the computed reliability, so adding one in good faith is not recklessness.
No sight of the coupling the spare brings, and no trace of the cause it shares, because the point of the addition is the gap it closes and not the paths it opens.

## Orders

Add another.
See the single point, put a backup behind it, compute the parallel reliability, and take the higher figure for a safer system, because the gap is real and the spare covers it.
Trust the added chance to raise the reliability, since to the designer it plainly does.

## Loses

The reliability it meant to raise, exactly by adding the part it was sure could only add.
Because the seat sees the gap the backup closes and not the interactions it opens, the cause it shares, or the watch it relaxes, it takes the climbing figure for the true reliability, so the redundancy meant to defend the system couples it, and the added chance that the arithmetic counted independent is felled with the rest by the one cause they share.
It gains a single point removed and a figure climbing, and loses the reliability it was sure the spare could only raise.

## Drives

Toward the backup added and the figure taken for the reliability,
the gap closed on the page felt as safety in the system,
so the seat that adds another puts a spare behind the primary, and takes the higher number for the safer system it never checked was coupled.
