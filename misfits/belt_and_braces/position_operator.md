# Position: Operator

## Taxonomy

[Belt and Braces](play_belt_and_braces.md). The seat that runs the system knowing a backup exists, where a real reserve is spent as room and the watch is shared until it is dropped.

## Owner

- Project: khai-misfits

## Has

A system to run and a backup behind it: the controls in hand and the spare that stands ready if the primary fails.
A real reserve: the backup does provide a margin, so working within it is not negligence but use of a genuine reserve.
No way to know the reserve is shared, and no signal that the watch each shares is a watch each is dropping, because the net is advertised and the diffusion is not.

## Orders

Lean on the spare.
Run the system trusting the backup to catch what you miss, share the watch with the guard beside you, and use the margin the reserve provides, because the backup is there and the margin is real.
Take the risk the net allows, since a covered risk is a smaller risk.

## Loses

The vigilance and the margin the backup was to add, exactly by trusting it.
Because a guard stands behind each guard, the seat lets responsibility for the catch diffuse across the set and watches assuming the other watches, and because it knows a net exists it runs closer to the edge, so the reserve the backup added is spent back as speed or load or ease, the system sitting as near the fall as before but with more parts to fail.
It gains the use of a real margin, and loses the safety the margin was, spent back into risk.

## Drives

Toward the reserve used and the watch dropped,
the net advertised felt as room while the diffusion goes unseen,
so the seat that leans on the spare runs closer to the edge, and spends the added safety back into the risk the backup seemed to cover.
