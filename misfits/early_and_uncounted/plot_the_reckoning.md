# Plot: The Reckoning

## Taxonomy

[Early and Uncounted](play_early_and_uncounted.md)

## Owner

- Project: khai-misfits

## Cue

[The Thin Years](place_the_thin_years.md), a trait's late face showing itself at last.
A cost arrives in whatever remains of the cohort, and a tally that must weigh it exactly as narrowly as it reaches.

## Action

[Weighing](process_weighing.md): the Count applies the identical procedure it always has, tallying the Trait's late effect against the Remnant present, and returns almost nothing, however severe the cost.

## Stage

[The Count](persona_the_count.md) acts from [the Tally](position_the_tally.md) on [Weigh What's Left](plan_weigh_whats_left.md), returning [the thin count](piece_the_thin_count.md) the Remnant's true size warrants, the discount correct and the harm to those few unrecorded by it.
The reckoning arrives exactly as the rule requires, no exception made and none needed.

## Tension

The discount is fair, and the fairness is what makes it inescapable.
There is no error: the Count weighs exactly what is present, and a remnant this thin can only return a weight this small, so the reckoning is the tally doing precisely its job, the same job it did in the Thick Years. But the fairness that correctly discounted a small population is now discounting a real, sometimes severe cost, so the plot closes with a late effect passed unselected against, not through any oversight but because the rule, honestly applied, has almost no one left to weigh it with.
