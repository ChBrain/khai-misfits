# Plan: Set It Low

## Taxonomy

A **standing plan**: the intent of [the setter's seat](position_setter.md), inside [Alarm Fatigue](play_alarm_fatigue.md).

## Owner

- Owner: [Setter](position_setter.md)

## Direction

Set the threshold low so no danger is missed. Warn on anything that might be the thing, and accept the false cries as the cheap price of catching the deadly rare one. For a single alarm the false alarm costs little and the missed danger costs everything, so err toward the cry.

## Orders

([The Setter](persona_the_setter.md) sets the alarm to trip on the merest sign, weighing the cheap false cry against the dreadful missed danger and erring, rightly for one alarm, toward the warning.)

## Implementation

([The Panel](place_the_panel.md); the [warning](process_warning.md) process; [the threshold](piece_the_threshold.md) is set low, so the alarm cries on the merest sign and, against a benign world, mostly raises [the false alarm](piece_the_false_alarm.md).)
