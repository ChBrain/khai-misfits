# Plot: The Miss

## Taxonomy

[Alarm Fatigue](play_alarm_fatigue.md)

## Owner

- Project: khai-misfits

## Cue

[The Floor](place_the_floor.md), the real one coming at last, in the flood of false.
The true reading rising, honest and unremarkable-looking, in a channel already saturated and a responder already deadened.

## Action

[Deadening](process_deadening.md): [The Wolf](persona_the_wolf.md), holding [the wolf's seat](position_wolf.md), trips the alarm truly, while [The Responder](persona_the_responder.md), holding [the responder's seat](position_responder.md), silences it exactly as it has silenced the hundred false cries before.

## Stage

[The Wolf](persona_the_wolf.md), on [Slip Through](plan_slip_through.md), arrives rarely and truly and sets off the alarm in the very voice worn out on nothing.
[The Responder](persona_the_responder.md), still on [Tune It Out](plan_tune_it_out.md), meets the true cry with the trained doubt the flood taught, and silences the one that mattered with the rest.

## Tension

The danger the alarm was set to catch is the one it causes to be missed.
There is still no villain: the wolf sounds honestly, the alarm cries faithfully, and the responder discounts as a hundred false alarms have rightly taught it to. But the true cry is voiced no differently from the false and lands on an ear taught to doubt them all, so the plot closes on the exact outcome the low setting was meant to prevent, the real event unanswered, missed not for want of an alarm but because the alarm cried too often for nothing.
