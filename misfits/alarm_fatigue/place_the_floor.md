# Place: The Floor

## Taxonomy

[Alarm Fatigue](play_alarm_fatigue.md). Where the danger is real and the responder works, the ward or control room the true event finally arrives on.

## Owner

- Project: khai-misfits

## Shown

The place the alarm is meant to guard, beds or gauges or instruments, where the harmless conditions and the deadly one both actually occur.
The responder at its work amid the running din, and, at last, the true reading rising, real and unremarkable-looking, in the flood of false.

## Holds

The deadening met by the wolf, the trained silence and the one cry that should have broken it.
Here the responder's learned discounting meets the real event, so the floor holds the point where a rational habit becomes a fatal one: the true alarm sounds exactly like the false and lands on a responder taught to doubt them all, so it is where the danger the alarm was set to catch is met with the silence the alarm itself trained.

## Offers

The real event, plainly and honestly, to whoever can still hear it.
Because the floor is where the danger actually happens, it offers the responder a true alarm that triggers as clearly as any drill, so it hands out the one cry that matters, and hands it out in the same worn voice and to the same deadened ear as all the false ones before it.

## Withheld

A second chance to tell true from false.
The floor cannot make the real event announce itself as different, so it withholds any tell that this cry is the one, and, the responder already taught to discount, it withholds the fresh attention the true alarm would need, so the danger passes not unsounded but unheard.
