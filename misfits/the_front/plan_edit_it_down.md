# Plan: Edit It Down

## Taxonomy

A **standing plan**: the intent of [the withholder's seat](position_withholder.md), inside [The Front](play_the_front.md).

## Owner

- Owner: [Withholder](position_withholder.md)

## Direction

Set the standing it would cost to show the unresolved thing against the gain of showing it, which the room does not give. Reckon honestly: while the thing is still raw it reads as noise and costs real standing, so edit it down toward the presentable, or, when it will not go, hold it.

## Orders

([The Burning](persona_the_burning.md) weighs the raw signal against how it will read, finds it will land as a colleague not yet on top of it, and edits it down or holds it, the burning part kept back exactly when the group most needs it shown.)

## Implementation

([The Green Room](place_the_green_room.md); the [editing](process_editing.md) process turned on one's own signal; [the raw signal](piece_the_raw_signal.md) weighed against [the premium](piece_the_premium.md) and left behind [the curtain](piece_the_curtain.md).)

## Targets

- [ ] the standing it would cost to show the raw is set against the gain, which the room does not give
- [ ] the raw is found to read as noise while it is still unresolved
- [ ] the burning thing is edited down toward the presentable, or held when it will not go
- [ ] the cheaper side is chosen, which is to keep the signal back until it is no longer the signal
