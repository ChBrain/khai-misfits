# Position: The Assay

## Taxonomy

[The Best on Paper](play_the_best_on_paper.md). The valuation seat, where each option is appraised on its own and returned as a single unbiased estimate.

## Owner

- Project: khai-misfits

## Has

An option to value, and a remit that ends at the number.
A per-option charge: the seat appraises one thing at a time on its own merits, blind to the others and to the ranking to come, so its whole job is a single honest estimate of a single option's worth.
No say in what is done with the estimate, since the seat produces a reading and hands it on, with no jurisdiction over how the readings are compared or which is taken.

## Orders

Value each fairly.
Appraise the option in front of the seat as accurately as it can be appraised, leaving the estimate as likely to fall short as to overshoot, so no single valuation carries a slant.
Report the honest number and pass it on, because the seat's duty is accuracy on each option alone and nothing beyond the one reading it owes.

## Loses

Authority over the aggregate, which the seat never held.
The Assay values each option in isolation and cannot see that its honest estimates will be ranked, so it produces symmetric, blameless errors with no way to know that a maximum will later condition on the ones that ran high.
What the seat forfeits is any governance of the selection: confined to the single reading, it cannot register that its unbiased numbers become a biased choice once the top is taken, so the seat that is right about every option one at a time has no lever over the aggregate its readings feed.

## Drives

Toward a slate of honest, isolated estimates,
and no hand on the ranking that will bias them,
so the seat that values each option truly is the one whose true valuations the selection downstream turns against the winner.
