# Plot: Proxy

## Taxonomy

[Metric Fixation](play_metric_fixation.md)

## Owner

- Project: khai-misfits

## Cue

[The charter](place_the_charter.md), where the goal is written large and cannot be counted.
The organisation wants the goal managed, and to manage it, it must measure it. At [the definition](place_the_definition.md), a stand-in is about to be chosen.

## Action

[Operationalisation](process_operationalisation.md): [The Proxy](persona_the_proxy.md), holding [the metric author's seat](position_metric_author.md), reduces the goal to a measure that tracks it today.
Then [targeting](process_targeting.md): [The Quota](persona_the_quota.md) at [the target setter's seat](position_target_setter.md) turns that measure into a number every desk must hit.

## Stage

[The Aim](persona_the_aim.md), holding [the mission owner's seat](position_mission_owner.md), tables [Serve the Purpose](plan_serve_the_purpose.md): keep the actual goal in view, the one plan with no number to defend itself.
[The Proxy](persona_the_proxy.md) tables [Make it Measurable](plan_make_it_measurable.md): you cannot manage what you cannot measure, and produces [the proxy](piece_the_proxy.md).
[The Quota](persona_the_quota.md) tables [Set the Target](plan_set_the_target.md): give the measure a bar, and [the target](piece_the_target.md) is set.

## Tension

Nothing here is a lie.
The goal is real and unmeasurable, the proxy really does track it today, and a target really is how organisations get things done. The room turns to the number because the number is the only thing it can hold, and the goal drops off the dashboard by a firm doing exactly what accountable management asks.
