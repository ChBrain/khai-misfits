# Process: Appraising

## Taxonomy

[The Winning Bid](play_the_winning_bid.md). Estimating the hidden value of the prize.

## Owner

- Project: khai-misfits

## Initiated by

A prize of common value going up for sale.
The moment the contest opens, every bidder must arrive at a number.

## Direction

From evidence toward an estimate.
It moves to a defensible value, worked up honestly from the signals available.

## Lever

Analysis meeting an unknown truth.
The force is careful method; the resistance is that the value is hidden, so the number lands around it with an error no one can sign.

## Echo

A field of honest estimates.
It hands forward a set of guesses scattered about the truth, whose highest the auction will single out.
