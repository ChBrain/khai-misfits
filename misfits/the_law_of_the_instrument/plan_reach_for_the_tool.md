# Plan: Reach for the Tool

## Taxonomy

A **standing plan**: the intent of [the wielder's seat](position_wielder.md), inside [The Law of the Instrument](play_the_law_of_the_instrument.md).

## Owner

- Owner: [Wielder](position_wielder.md)

## Direction

When a problem appears, reach for the tool you have mastered. See it in the instrument's terms, promote the face you can strike to the whole of it, and work, because you have solved a hundred problems just like this and the reflex is the skill. Do not stop to ask what the problem is: you already know how it will be solved.

## Orders

([The Wielder](persona_the_wielder.md) reaches for the mastered tool before diagnosing, and solves the version of the problem the tool fits.)

## Implementation

([The Workshop](place_the_workshop.md); the [reaching](process_reaching.md) process; [the reflex](piece_the_reflex.md) fires ahead of any diagnosis, so [the tool](piece_the_tool.md) is chosen before the problem is understood.)

## Targets

- [ ] the mastered tool is reached for the moment a problem appears
- [ ] the problem is seen in the instrument's terms, not its own
- [ ] the strikeable face is promoted to the whole of the problem
- [ ] the reflex fires before the diagnosis, and the skill becomes the blindness
