# Plot: The Draught

## Taxonomy

[Model Collapse](play_model_collapse.md)

## Owner

- Project: khai-misfits

## Cue

[The Corpus](place_the_corpus.md), the next builder needing data and the cheap pool right there.
The common now part synthetic, and a fresh model to train on whatever is abundant.

## Action

[Recursing](process_recursing.md): [The Trainer](persona_the_trainer.md), holding [the trainer's seat](position_trainer.md), draws again from the pool, while [The Pool](persona_the_pool.md), holding [the pool's seat](position_pool.md), hands back an ocean already an echo.

## Stage

[The Trainer](persona_the_trainer.md), still on [Train on What's There](plan_train_on_whats_there.md), draws the abundant cheap data for the next build, sensibly, unable to tell how much is already model output.
[The Pool](persona_the_pool.md), still on [Take It All In](plan_take_it_all_in.md), offers its water as one undifferentiated ocean, richer in echo and poorer in the tails than the last draught.

## Tension

The sensible draught is drawn from a well already gone part echo.
There is still nothing to fault: the trainer draws the abundant data at hand as any builder would, and the pool offers what it holds. But the water is already thinned and unmarked, so the plot closes with a new generation trained on a world narrowed by the last, faithful to it, and set to narrow it again.
