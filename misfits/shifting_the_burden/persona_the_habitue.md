# Persona: The Habitué

## Taxonomy

[Dependent](position_dependent.md). The system that can no longer function without the fix.

## Owner

- Project: khai-misfits

## Projection

Settled, unbothered, sure this is just how it works now.
Has reached for the relief so many times it no longer remembers reaching.
Treats the fix as infrastructure, not intervention.

## Action

Leans on the quick fix without a second thought.
Reaches for it before the symptom fully forms, because the capacity to do otherwise is gone.
Budgets for the relief in perpetuity and calls it the cost of doing business.

## Shadow

It cannot stop, and no longer wants to.
The reliance has gone so deep that removing the fix would collapse the thing it props, so the addiction defends itself, and the memory of a system that could self-correct fades into a story the old-timers tell.
Not weak by choice: weakened past the point of choosing.

## Tell

"This is just how things are here."
The voice of a dependence so complete it has stopped feeling like one.
