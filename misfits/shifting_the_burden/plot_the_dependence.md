# Plot: The Dependence

## Taxonomy

[Shifting the Burden](play_shifting_the_burden.md)

## Owner

- Project: khai-misfits

## Cue

[The dispensary](place_the_dispensary.md), and a symptom returning to a capacity too wasted to cure it.
The reach for the fix has become reflex, made before the flare has even fully formed.

## Action

[Depending](process_depending.md): [The Habitué](persona_the_habitue.md), holding [the dependent's seat](position_dependent.md), leans on the fix without a second thought, because the fix is now the only thing that still works.

## Stage

[The Habitué](persona_the_habitue.md) tables [Lean on the Fix](plan_lean_on_the_fix.md): keep the relief flowing, budget for it forever, build around it, so [the dependence](piece_the_dependence.md) hardens into infrastructure.
The cure is no longer merely deferred but unreachable, defended by everything now resting on the relief, and the memory of a system that could self-correct fades into a story.

## Tension

The choice is gone, and its absence goes unnoticed.
There is no drama here, only settling: the fix is treated as how things simply work, the reliance has stopped registering as reliance, and that is exactly the state that makes it unbreakable. The system is not deciding to depend on the fix. It has been worn past the point where depending or not is still a thing it can decide.
