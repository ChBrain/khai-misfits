# Place: The Floor

## Taxonomy

[The Lowest Bar](play_the_lowest_bar.md). Where a common minimum would be set, the table at which the jurisdictions would agree a floor below which none may cut, the one arrangement that holds the bar against the base's mobility.

## Owner

- Project: khai-misfits

## Shown

The place the jurisdictions would agree a common minimum, a treaty table, a harmonisation, a shared standard below which none may register a base.
The Bound calling for the floor and the jurisdictions weighing it, each knowing it must forgo the base it could win by cutting below the line they would draw.

## Holds

The floor, and the defection that threatens it.
Here a common minimum is set for all rather than a bar set by each, so the Floor holds the point where the descent could stop: a standard no jurisdiction may cut beneath, so the base cannot be drawn by going lower and the immobile party keeps its protection, so it is where the standard is grounded in a shared line rather than in the terms the base prefers, the one arrangement that holds the bar against the mobility that lowered it.
The floor holds only while all keep it, and each retains the power to win the base back by breaking it.

## Offers

A way to stop the descent, and the immobile party's protection kept.
Because the Floor sets a minimum below which none may cut, it offers the one thing the Market of Laws could not, a bar the base cannot slip beneath, so it hands out the descent halted and the protection held, and hands it out at the price of every jurisdiction forgoing the base it could win by defecting, a floor kept only by all continuing to pay.
The protection is real while the floor holds, and holding it costs every jurisdiction a base it could take.

## Withheld

Any floor that holds itself.
The Floor can set a common minimum but cannot make it self-enforcing, because any jurisdiction that quietly cuts below draws the base and the revenue back, so it withholds the floor that needs no continuing cost, exacting from each the standing forbearance of not undercutting, and, all shared minimum and no self-enforcement, it withholds any guarantee against the defection that unravels it, the floor lasting only as long as every jurisdiction keeps paying to hold it.
