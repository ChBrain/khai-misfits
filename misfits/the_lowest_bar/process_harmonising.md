# Process: Harmonising

## Taxonomy

[The Lowest Bar](play_the_lowest_bar.md). The Bound and the jurisdictions setting a common minimum below which none may cut, the costly grounding that holds the bar against the base's mobility and never enforces itself.

## Owner

- Project: khai-misfits

## Initiated by

A descent toward the lowest bar and a party who cannot follow the base out.
At [the floor](place_the_floor.md), the jurisdictions agreeing a common minimum below which none may register a base, called for by the immobile party the descent lands on.

## Direction

Toward the floor set and the base held above it.
Because the descent runs on the base being drawable to a lower bar, harmonising sets a minimum below which none may cut, so the base cannot be won by going lower and the immobile party keeps its protection, the standard grounded in a shared line rather than in the terms the base prefers, and the descent halted short of the bottom.
The floor stops the race only while every jurisdiction keeps it.

## Lever

The willingness of each jurisdiction to forgo the base it could win by defecting.
Because any jurisdiction that quietly cuts below the floor draws the base and the revenue back, harmonising runs only on each choosing to forgo that base, so the remedy is held continuously and never for free, every jurisdiction carrying a standing incentive to defect and the floor lasting only as long as all keep paying to hold it.

## Echo

A floor agreed, and a defection always available beneath it.
The jurisdictions have set a minimum the base cannot slip beneath, so what reads as the descent stopped is, in truth, a floor held only by continuing forbearance, the protection kept while all pay and gone the moment one jurisdiction cuts below to draw the base back.
