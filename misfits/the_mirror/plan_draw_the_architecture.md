# Plan: Draw the Architecture

## Taxonomy

A **standing plan**: the intent of [the architect's seat](position_architect.md), inside [The Mirror](play_the_mirror.md).

## Owner

- Owner: [Architect](position_architect.md)

## Direction

Specify the clean design the problem deserves: low coupling, high cohesion, narrow explicit interfaces. Draw the seams where the architecture wants them, and assume the teams will honour them.

## Orders

([The Drafter](persona_the_drafter.md) decomposes the system as if any team could build any module, and hands the shape down.)

## Implementation

([The Whiteboard](place_the_whiteboard.md); the [designing](process_designing.md) process; a decomposition chosen for the problem, blind to the communication structure that will build it.)

## Targets

- [ ] the system is decomposed for low coupling and high cohesion
- [ ] interfaces are drawn narrow and explicit
- [ ] the seams are placed where the design wants them
- [ ] the shape is specified as though teams were interchangeable
