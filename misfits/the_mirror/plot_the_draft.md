# Plot: The Draft

## Taxonomy

[The Mirror](play_the_mirror.md)

## Owner

- Project: khai-misfits

## Cue

[The whiteboard](place_the_whiteboard.md), clean light on clean boxes.
The architecture is being drawn: modules, interfaces, low coupling, a design that would work if the teams were interchangeable. The org chart is not in the room.

## Action

[Designing](process_designing.md): [The Drafter](persona_the_drafter.md), holding [the architect's seat](position_architect.md), decomposes the system for the problem and places the seams where the design wants them.

## Stage

[The Drafter](persona_the_drafter.md) tables [Draw the Architecture](plan_draw_the_architecture.md): specify as though any team could build any module, so [the architecture](piece_the_architecture.md) is set clean and [the interface](piece_the_interface.md) is drawn narrow and explicit.
Every edge on the board looks equally sound, including the one that will run between two teams who never talk.

## Tension

Nothing here is a mistake.
The design is genuinely good, and it is good for an org that does not exist: an interchangeable one, where any team could own any module. The seams are placed where the problem wants them, on a board that cannot show where the org already put its walls, so the cleanest edge drawn is the one that will fray first.
