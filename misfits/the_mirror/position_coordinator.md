# Position: Coordinator

## Taxonomy

[The Mirror](play_the_mirror.md). The seat that carries communication across the boundary the org made expensive to cross.

## Owner

- Project: khai-misfits

## Has

A standing invitation to both teams, and a mandate to align them.
The tools of coordination: the sync, the shared doc, the translation between two contexts.

## Orders

Run the cross-team meeting and keep the interface aligned.
Carry each team's context to the other, and file the shared decision.
Treat the seam as a communication problem, and communicate.

## Loses

The bandwidth a single shared owner would have had for free.
A seat that can only pass messages across the boundary cannot make two teams into one, so the alignment it wins is always thinner than the problem.

## Drives

Toward a narrowed but unclosed gap, and away from the co-ownership that would have closed it.
Toward more meetings against a seam that meetings cannot move.
