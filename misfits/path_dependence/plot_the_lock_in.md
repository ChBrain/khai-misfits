# Plot: The Lock-In

## Taxonomy

[Path Dependence](play_path_dependence.md)

## Owner

- Project: khai-misfits

## Cue

[The rut](place_the_rut.md), the standard now built into the tools, the training, the files, the habits.
The groove is worn so deep that the wheels follow it without steering, and climbing out is a thing no one can do alone.

## Action

[Locking](process_locking.md): [The Captive](persona_the_captive.md), holding [the captive's seat](position_captive.md), would take the better option but weighs a solo switch and stays, while [The Incumbent](persona_the_incumbent.md), holding [the incumbent's seat](position_incumbent.md), need do nothing but be the groove everyone is already in.

## Stage

[The Captive](persona_the_captive.md) tables [Wait for the Others](plan_wait_for_the_others.md), because [the switching cost](piece_the_switching_cost.md) makes moving alone a loss, so [the lock-in](piece_the_lock_in.md) is held by every captive's cost to leave.
[The Incumbent](persona_the_incumbent.md) tables [Ride the Lead](plan_ride_the_lead.md), secure in a standard kept by inertia and not by any fresh choice to keep it.

## Tension

Everyone would rather be elsewhere, and everyone rationally stays.
Each captive prefers the better option and each is right that going first would strand him, so the standard is held in place by a crowd that would each leave if the others did, and none of whom can afford to lead. No one chose to keep it; they only each declined to pay, alone, the cost of escape, and the sum of those declinings is a lock nobody holds and no one can open.
