# Persona: The Captive

## Taxonomy

[Captive](position_captive.md). The user locked into the standard, who would switch if everyone did, and cannot move alone.

## Owner

- Project: khai-misfits

## Projection

Invested, encumbered, quietly aware there is better.
Has the tools, the training, the files, the habits all built around the standard.
Would take the better option gladly, if only the whole crowd would take it too.

## Action

Waits for the others, and stays put.
Weighs the cost of switching alone, the lost investment and the stranding from the base, against a gain that only arrives if everyone moves, and rationally does not move.
Holds to the standard not because it is best but because leaving it by himself would cost more than it is worth.

## Shadow

Every captive reasons the same, so no one goes first and all remain.
The lock is not a lock anyone holds but the sum of each user's cost to leave, so the standard is kept by a crowd that would each prefer the better one and none of whom can afford to be the first to it.
Not trapped by force: held in place by a switching cost that only moving together could pay.

## Tell

"I'd switch in a heartbeat, if everyone else would."
The universal condition that keeps everyone exactly where they are.
