# Position: Incumbent

## Taxonomy

[Path Dependence](play_path_dependence.md). The seat of the leading standard, that wins by growing.

## Owner

- Project: khai-misfits

## Has

An early lead, however it was won.
Increasing returns that widen it, a bigger base and a lower cost with every adopter.
No need to be the best, only to stay in front.

## Orders

Ride the lead.
Let scale and the network do the competing.
Grow, and let growth be its own argument.

## Loses

Nothing, so long as no one can coordinate against it.
Its position is secured by the very base it built, and a superior rival cannot take that base one user at a time, so the seat need never improve to keep winning.
Held aloft by a loop that rewards being ahead over being good.

## Drives

Toward an ever-larger base,
away from any contest on the merits,
so a standard that got ahead first stays ahead without ever proving it deserved to.
