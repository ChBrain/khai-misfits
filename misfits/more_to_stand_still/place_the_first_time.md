# Place: The First Time

## Taxonomy

[More to Stand Still](play_more_to_stand_still.md). The peak, where the motivator lands with its full and unadapted force, the height the first dose gives and no later dose at the same amount will reach again.

## Owner

- Project: khai-misfits

## Shown

A motivator working at full strength, the effect at its brightest.
Shown as promise: the dose lands and the response leaps, the result so strong that this looks like the natural power of the motivator, the amount and the effect matched at their best and seeming to set the rate for all that follow.

## Holds

Holds the unadapted response, the one time the level has not yet caught up to the dose.
The First Time is where the effect is largest because the neutral is lowest, the stimulus measured against a baseline it has not yet raised, so the same amount that will later register as nothing here registers as everything, the peak a function not of the dose but of the distance it has from a zero not yet moved.

## Offers

Offers the full effect once, and the standard that will make every repeat a shortfall.
Here the motivator delivers everything it can, so the First Time offers a true and complete result, and by being the brightest it also offers the mark against which every later dose is judged, so the peak that thrills becomes the memory the climb is chasing, each raise measured against a height that only an unadapted response could reach.

## Withheld

Withholds that the height came from the low baseline, not the dose.
What the First Time hides is that its power was borrowed from a neutral not yet raised, so the effect will never be this large again at this amount, not because the dose weakened but because the response adjusted, the peak concealing that it was the one reading taken before adaptation began and so the one that can never be repeated except by climbing.
