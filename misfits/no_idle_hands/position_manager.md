# Position: Manager

## Taxonomy

[No Idle Hands](play_no_idle_hands.md). The seat that prices idle capacity as waste and drives the busy fraction up, where closing the gaps reads as leanness gained and the line behind the server is out of sight.

## Owner

- Project: khai-misfits

## Has

A resource with a fixed cost and a metric that reads its busy fraction: capacity paid for whether it works or not, and utilisation to be raised.
A sound aim: the loading is rational, idle capacity genuinely costs and closing the gaps wrings more output from the same price.
No sight of the line, which forms behind the server and not at it, and no sign that the idle time being spent is the buffer the flow needs.

## Orders

Drive out the idle time.
Feed the server the next job the moment it frees, close every gap, and raise the busy fraction point by point toward one.
Read a high utilisation as good, because an idle hour is capacity bought and thrown away.

## Loses

The chance of seeing that the idle time is a buffer, exactly because the metric reads utilisation and not the wait.
Because the line forms behind the server where the utilisation metric does not look, the seat sees the resource busy and cannot see the queue swelling, so drives the last increments hardest for looking most virtuous and spends the slack that absorbed the variability, losing the flow to its own leanness until the choke.
It gains more output from the same fixed cost, and loses the buffer that kept the wait bounded.

## Drives

Toward the idle time driven out and the busy fraction raised,
the resource kept fully occupied and read as lean,
so the seat that prices idleness as waste spends the buffer point by point, sure to the choke that a fully occupied resource is the leanest.
