# Place: The Floor

## Taxonomy

[No Idle Hands](play_no_idle_hands.md). Where the server works and the busy fraction is watched, the place the idle time is priced as waste and driven out and the line behind the server goes unseen.

## Owner

- Project: khai-misfits

## Shown

The place the resource is loaded and its utilisation read, a plant floor, a ward, a desk, a scheduler's board, where the metric reads the busy fraction and reads it high.
The Manager watching the gaps and closing them, the Server taking the next job the moment it frees, and everyone seeing a resource kept fully occupied and calling it lean.

## Holds

The busy fraction, and the slack it spends.
Here the server is loaded and the metric watched, so the Floor holds the point where idleness is priced and erased: an idle hour read as waste and driven out, the busy fraction raised point by point, so it is where the loading is done and the leanness is felt, and where the slack that absorbed the variability is spent for a metric that reads better with every gap closed.

## Offers

A resource fully occupied and a metric that reads well.
Because the Floor is where the busy fraction is watched and raised, it offers a high utilisation and the look of a flow run lean, the gaps closed and the output wrung from the fixed cost, so it hands out the leanness of a busy resource, and hands it out on the busy fraction and not the wait, the line that grows behind the server left out of what it shows.

## Withheld

Any sight of the line the loading grows.
The Floor shows the server busy but cannot show the queue, because the wait forms behind the work and the metric reads only utilisation, so it withholds the swelling line beneath the high fraction, letting the leanness of the server stand for the health of the flow, and, all busy resource and no visible wait, it withholds the cost of the last increments until the backlog diverges at the wall.
