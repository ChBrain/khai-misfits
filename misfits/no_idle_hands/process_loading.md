# Process: Loading

## Taxonomy

[No Idle Hands](play_no_idle_hands.md). The Manager driving the busy fraction up by squeezing out the idle time, the rational close of every gap that spends the buffer the flow needed.

## Owner

- Project: khai-misfits

## Initiated by

Idle capacity priced as waste.
At [the floor](place_the_floor.md), the manager seeing the gaps between jobs and feeding the server the next the moment it comes free.

## Direction

Toward the idle time driven out and the busy fraction raised.
Because a fixed cost runs whether the resource works or not, loading closes the gaps to wring more output from the same price, so the busy fraction climbs point by point toward one and each closed gap reads as leanness gained, the idle time that was the buffer for the variability spent without ever being seen as a buffer.

## Lever

The real cost of an idle hour.
Because capacity paid for and unused genuinely is a cost, loading runs not on any greed but on the plain sense of erasing waste, so the buffer is spent by good management doing its job, the slack that absorbed the variability squeezed out because it looked, exactly, like the waste a manager is right to hate.

## Echo

The gaps closed, and the buffer spent for a metric.
The manager has raised the busy fraction sensibly, so what reads as leanness gained is, in truth, the idle time the flow needed to absorb its variability, driven out because it cost, the variability it held moved to the line behind the server where the metric does not look.
