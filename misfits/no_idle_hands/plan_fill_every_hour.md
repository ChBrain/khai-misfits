# Plan: Fill Every Hour

## Taxonomy

A **standing plan**: the intent of [the manager's seat](position_manager.md), inside [No Idle Hands](play_no_idle_hands.md).

## Owner

- Owner: [Manager](position_manager.md)

## Direction

Price the idle time as waste and drive it out. Feed the server the next job the moment it comes free, close every gap, and raise the busy fraction point by point toward one, because a fixed cost runs whether the resource works or not and each closed gap wrings more output from the same price.

## Orders

([The Manager](persona_the_manager.md) loads the server to close the gaps, reads a high utilisation as leanness gained, and spends the slack that absorbed the variability without seeing it was a buffer.)

## Implementation

([The Floor](place_the_floor.md), where the busy fraction is watched; the [loading](process_loading.md) process; [the utilisation](piece_the_utilisation.md) is driven up by spending [the slack](piece_the_slack.md), the buffer that held [the variability](piece_the_variability.md) the line behind the server now must hold.)

## Targets

- [ ] the idle time is priced as waste and the gaps are closed
- [ ] the server is fed the next job the moment it frees
- [ ] the busy fraction is driven up point by point toward one
- [ ] the slack that absorbed the variability is spent as the metric reads lean
