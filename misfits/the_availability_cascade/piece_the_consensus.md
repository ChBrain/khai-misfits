# Piece: The Consensus

## Taxonomy

[The Availability Cascade](play_the_availability_cascade.md). The visible shared alarm each person updates on and echoes, the swelling agreement that mixes belief and silence and cannot tell them apart.

## Owner

- Project: khai-misfits

## Place

Builds in [the forum](place_the_forum.md), where each sees how alarmed everyone else appears.
Carried to [the chamber](place_the_chamber.md), where its size becomes the mandate the response must meet.

## Load Bearing

Without it the fear would stay one mind's, and a vivid case would frighten each person alone with no way to compound.
With it the fear becomes a crowd's: because each takes the shared alarm as evidence and updates toward it, and each echoes it rather than pay the cost of lone dissent, the Consensus swells with belief and with silence at once, so its size is read as its proof. The Consensus is load bearing because it carries both cascades (Kuran and Sunstein): the informational, in which the shared belief is taken as evidence, and the reputational, in which the safe move is to echo, so the piece holds the swelling agreement whose growth is self-feeding and whose composition, conviction or mere quiet, can never be seen.

## Apparent

A public that has weighed the danger and agreed, the reassuring sense that so many cannot be wrong.
Seen at the surface, the Consensus looks like collective judgement, a shared conclusion many minds have reached, and the truth that its size measures how visibly it is voiced and not how independently it is held, that the voices adding belief cannot be told from the voices only withholding doubt, stays hidden behind how convincing a broad agreement plainly seems.

## Yearbook

Built in [the forum](place_the_forum.md) through [cascading](process_cascading.md), the shared alarm each of [the public](persona_the_public.md) updates on and swells.
It is the crowd's part of the trap: the Consensus turns the felt fear of [the salience](piece_the_salience.md) into a shared certainty, its size becoming the mandate that draws [the overreach](piece_the_overreach.md), the swelling twin of the silence it hides, climbing through [the availability cascade](piece_the_availability_cascade.md).
